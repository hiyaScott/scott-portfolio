💡 项目概述

"妈妈！"——这是每个家庭里最常出现的声音之一。小朋友们似乎有无尽的"妈妈"要喊，有时候一天能喊上百次。很多家长都好奇：我的孩子一天到底喊多少次妈妈？

核心硬件: ESP32-Display (1.28寸圆形LCD) + MAX9814麦克风

屏幕规格: 240×240分辨率, GC9A01驱动, 65K彩色

目标续航: 24小时+ (500mAh锂电池)

⚡ 工作原理

设备通过声音检测算法识别"妈妈"这个词。当检测到符合条件的音频特征时，计数器+1，并在1.28寸圆形屏幕上实时显示。

声音检测流程: ┌──────────────┐ ┌──────────────┐ ┌──────────────┐ │ 麦克风采样 │ ──→ │ 峰峰值检测 │ ──→ │ 阈值判断 │ └──────────────┘ └──────────────┘ └──────┬───────┘ │ 否 │ 是 ┌──────────────────────────────┘ ↓ ┌──────────────┐ ┌──────────────┐ ┌──────────────┐ │ 更新波形图 │ ←── │ 计数器+1 │ ←── │ 触发冷却 │ └──────────────┘ └──────┬───────┘ └──────────────┘ │ ↓ ┌──────────────┐ │ 屏幕闪烁反馈 │ └──────────────┘

💻 代码架构

基于 Arduino + TFT_eSPI 库开发，主要功能模块:

// 核心函数 void setup() // 初始化屏幕、引脚、显示启动画面 void loop() // 主循环: 采样→检测→反馈→更新UI // 功能模块 int readMicrophone() // 读取麦克风峰峰值 void updateWaveform(int level) // 更新底部波形显示 void triggerMamaDetected() // 触发计数+视觉反馈 void drawCountDisplay() // 绘制中心计数显示 void handleButtonPress() // 按键处理(短按/长按)
