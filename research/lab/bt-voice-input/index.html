🎤 蓝牙语音输入器

解放双手的PC语音输入解决方案。通过蓝牙连接电脑，将语音实时转换为文字输入，让打字不再束缚你的双手。

🚀 启动 Web 模拟器 📋 查看设计文档

🎯 项目目标

蓝牙语音输入器是一个手持设备，通过蓝牙与PC连接，将用户的语音实时转换为文字输入。相比传统键盘输入，它能：

  • 解放双手，适合长时间文字输入场景
  • 提高输入效率，说话速度远高于打字速度
  • 降低打字带来的手腕疲劳
  • 支持多种输入模式（中文/英文/命令）

✨ 核心特点

🔌
蓝牙HID键盘
模拟标准USB键盘设备，无需安装驱动，即连即用。支持Windows、macOS、Linux等主流系统。
🎙️
一键录音
大按键设计，单手操作。按住录音，松开发送，交互简洁直观，零学习成本。
🌐
云端识别
接入专业语音识别API，准确率高，支持普通话、英语及多种方言。持续优化，越用越准。
🔋
超长续航
500mAh锂电池，支持连续录音6小时，待机超过30小时。USB-C快充，2小时充满。
📟
实时显示
0.96寸OLED屏幕，显示蓝牙连接状态、电池电量、输入模式。一目了然。
🔄
多模式切换
支持中文连续、英文输入、命令模式、静音模式。短按功能键快速切换。

🛠️ 硬件规格

组件 型号/规格 说明
主控芯片 ESP32-S3 双核240MHz，蓝牙5.0，支持BLE HID
麦克风 MSM261S4030H0R I2S数字麦克风，高灵敏度，低噪声
显示屏 0.96寸 OLED SSD1306驱动，128×64像素，I2C接口
电池 3.7V 500mAh 锂聚合物电池，支持USB-C充电
续航 约20小时 中等使用强度（每天录音1小时）
尺寸 80×40×15mm 手持遥控器大小，单手操作
重量 约60g 含电池，轻便易携带
