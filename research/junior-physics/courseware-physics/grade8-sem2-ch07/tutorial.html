实验简报 LAB BRIEFING — 船为什么能浮？
⚓ 引入问题

一艘万吨巨轮由钢铁制成，钢铁的密度约为 7.9×10³ kg/m³，远大于水的密度。然而，巨轮却能平稳地漂浮在海面上。与此同时，一枚小小的铁钉放入水中却会直接沉底。

问题：同样是铁，为什么巨轮能浮，铁钉会沉？究竟是什么力让船"克服"了重力？

🔬 本节实验目标
  1. 观察并描述物体在液体中所受向上托力的现象
  2. 理解浮力的方向始终为竖直向上
  3. 通过压力差分析浮力产生的原因
  4. 建立浮力与液体密度、排开体积的定量关系
核心思考题：如果将一个乒乓球按入水中，松手后它会迅速上浮——在这个过程中，是什么力在推动它？这个力的施力物体是什么？
02
浮力的概念 CONCEPT OF BUOYANCY — 定义·方向·成因
📐 定义

浮力（Buoyancy Force）：浸在液体（或气体）中的物体受到液体（或气体）对它向上的托力，叫做浮力。

符号：F浮
⬆️ 方向：竖直向上

浮力的方向总是竖直向上，与重力方向相反。

图示：浸没物体受力示意图 · 浮力与重力方向相反

💧 产生原因：液体压力差

物体浸入液体后，其各个表面都受到液体的压强。由于液体压强随深度增加而增大：

  • 下表面所处深度大 → 向上的压力 F向上 较大
  • 上表面所处深度小（或露出液面） → 向下的压力 F向下 较小
F浮 = F向上 − F向下

这就是浮力产生的根本原因——上下表面的压力差。

验证实验
压力差法感知浮力

器材：去底塑料瓶、乒乓球、水槽

步骤：

  1. 将去底塑料瓶瓶口朝下，放入乒乓球
  2. 向瓶中注水，观察乒乓球——不浮起（下表面无水，无向上压力）
  3. 用手堵住瓶口，乒乓球立即上浮（下表面受到水的向上压力）

结论：浮力的产生依赖于下表面受到的向上压力。没有向上的压力，就没有浮力。

⚠ 误差分析
  • 瓶口密封性：若瓶口与手掌贴合不严，水会渗漏，导致下表面受压不足，实验现象不明显
  • 乒乓球材质：部分乒乓球表面有细微孔洞，内部可能进水改变平均密度，影响浮起速度
  • 操作时机：堵瓶口动作需迅速果断，延迟会导致水从缝隙流失过多
  • 水位高度：瓶内水位需足够高以产生明显的上下压强差
03
阿基米德原理 ARCHIMEDES' PRINCIPLE — 定量计算浮力
🏛️ 历史背景

公元前3世纪，古希腊学者阿基米德在解决"王冠是否掺假"问题时，发现了流体静力学的基本规律——阿基米德原理。

相传他在浴缸中悟出道理，兴奋得赤身裸体冲上街道大喊"尤里卡！（我发现了！）"

📖 原理内容

浸在液体中的物体受到向上的浮力，浮力的大小等于物体排开的液体所受的重力。

F浮 = G排 = ρ液 · g · V排

ρ液 — 液体密度 (kg/m³)

g — 重力加速度 (9.8 N/kg)

V排 — 排开液体体积 (m³)

G排 — 排开液体重力 (N)

核心实验 A
称重法验证阿基米德原理

实验器材：弹簧测力计、金属块、烧杯、水、细线

实验步骤：

  1. 用细线拴好金属块，挂在弹簧测力计下，在空气中静止时记录示数 —— 即物体重力 G
  2. 将金属块缓慢浸入水中（不碰杯底/杯壁），记录此时弹簧测力计示数 F′
  3. 计算浮力：F浮 = G − F′
  4. 用量筒测出金属块排开水的体积 V排，计算排开水的重力 G排 = ρ水·g·V排
  5. 比较 F浮 与 G排，验证二者是否相等

⚠ 误差分析 · 称重法
  • 弹簧秤精度：学生用弹簧测力计分度值通常为0.1N或0.2N，读数时存在±0.05N的估读误差
  • 读数时机：金属块浸入水中后需等待水面稳定、测力计示数静止后再读数，抖动会导致读数偏大或偏小
  • 触底/触壁：若金属块接触烧杯底部或侧壁，杯壁会提供额外的支持力，导致F′偏小，计算的浮力偏大
  • 液体残留：金属块从空气中移入水中时表面可能附着气泡，改变实际排开体积
  • 细线体积：悬挂金属块的细线也排开少量水，精确测量时应扣除或忽略
核心实验 B
溢水杯实验 · 精确测量排开液体重力

实验器材：溢水杯、小桶（接水用）、弹簧测力计、金属块、水

实验步骤：

  1. 将溢水杯装满水至溢水口，确保水面与溢水口齐平（多余水自然流出）
  2. 用弹簧测力计测出空小桶的重力 G桶
  3. 测出金属块在空气中的重力 G
  4. 将金属块缓慢浸入溢水杯中，溢出的水全部流入小桶
  5. 测出桶+水总重力 G总
  6. 计算：排开水重力 G排 = G总 − G桶；浮力 F浮 = G − F′
  7. 比较 F浮 与 G排

⚠ 误差分析 · 溢水杯法
  • 初始水位：溢水杯未装满至溢水口，导致溢出水少于实际排开体积，G排偏小
  • 液体残留：溢水杯壁或漏斗表面附着水滴，未完全流入接水桶，导致测量的G排偏小
  • 读数时机：需等溢水完全停止后再收集，过早移走接水桶会遗漏部分溢出水
  • 金属块速度：浸入过快会导致液面波动、水花溅出，造成溢出水测量不准
  • 水的表面张力：溢水口处水膜可能阻碍溢流，轻微超装可减小此影响
关键记忆口诀："浮力等于排开重，液体密度乘gV" —— F浮 = ρ液gV排

物体的浮沉条件 FLOATING & SINKING — 密度决定命运
⚖️ 核心判据

物体的浮沉取决于浮力与重力的大小关系，也可通过物体密度与液体密度直接判断：

↑
漂浮
F浮 = G
ρ物 < ρ液
V排 < V物
=
悬浮
F浮 = G
ρ物 = ρ液
V排 = V物
↓
沉底
F浮 < G
ρ物 > ρ液
V排 = V物

密度对比图示：ρ物 vs ρ液 决定三种浮沉状态

📝 对比总结
状态 受力关系 密度关系 体积关系
漂浮 F浮 = G物 ρ物 < ρ液 V排 < V物
悬浮 F浮 = G物 ρ物 = ρ液 V排 = V物
沉底 F浮 < G物 ρ物 > ρ液 V排 = V物
观察实验
鸡蛋在盐水中的浮沉

器材：新鲜鸡蛋、玻璃杯、食盐、搅拌棒、水

  1. 将鸡蛋放入清水中，观察鸡蛋沉底（ρ蛋 > ρ水）
  2. 向水中逐渐加盐并搅拌，观察鸡蛋逐渐悬浮（ρ蛋 = ρ盐水）
  3. 继续加盐，鸡蛋最终漂浮（ρ蛋 < ρ盐水）

核心发现：通过改变液体密度，同一个鸡蛋可以在三种状态间转换。

⚠ 误差分析
  • 食盐溶解不均：搅拌不充分导致盐水密度分层，鸡蛋可能悬浮在不均匀的密度层中
  • 鸡蛋新鲜度：不新鲜鸡蛋内部气室增大，平均密度降低，可能在清水中就漂浮
  • 温度影响：水温影响水的密度（热胀冷缩），精密实验需记录水温
  • 加盐速度：过快加盐导致局部密度过高，鸡蛋被"顶起"而非真实漂浮
05
浮力的应用 APPLICATIONS — 从理论到工程
🚢 轮船 — 空心法增大排开体积

钢铁密度远大于水，但将钢铁做成空心结构后，整体平均密度大大降低。轮船排开的水量足以产生巨大的浮力来支撑船体和所载货物。

原理：做成空心 → V排增大 → F浮增大
排水量：轮船满载时排开水的质量。单位：吨（t）。排水量 = 船自重 + 载货量
🔱 潜水艇 — 改变自身重力

潜水艇通过水舱的充水/排水来改变自身重力，实现下潜与上浮：

  • 充水 → 重力增大 → G > F浮 → 下潜
  • 排水 → 重力减小 → G < F浮 → 上浮
  • 悬浮时：G = F浮（悬浮于任意深度）

注意：潜水艇体积基本不变，因此浮力大小不变，改变的是重力。

🎈 气球与飞艇 — 改变平均密度

热气球、氢气球和飞艇能够在空中上升，是因为内部填充气体的密度小于周围空气的密度：

热气球

加热空气 → 空气膨胀 → 密度降低 → ρ内 < ρ外 → 上升

氢气球/氦气球

填充密度极小的气体 → 整体平均密度 < 空气密度 → 上升

🌡️ 密度计 — 漂浮条件测液体密度

密度计在任何液体中都处于漂浮状态，因此 F浮 = G（恒定）。

根据阿基米德原理：

F浮 = ρ液·g·V排 = G计（恒定）

因此：ρ液越大 → V排越小 → 密度计浸入越浅

密度计的刻度上小下大，且刻度不均匀。

探究实验
自制简易密度计

器材：吸管、橡皮泥、刻度尺、盐水、清水

  1. 将橡皮泥密封在吸管的一端，使其能竖直漂浮在水中
  2. 在清水中标记液面对应的刻度（参考点）
  3. 放入不同浓度的盐水中，观察浸入深度的变化
  4. 根据浸入深度反推液体密度关系
⚠ 误差分析
  • 重心偏移：橡皮泥分布不均导致密度计倾斜，影响液面读数位置
  • 表面张力：细吸管受水的表面张力影响，液面呈弯月形，读数位置判定困难
  • 管壁厚度：吸管壁厚不可忽略，精确测量需考虑管壁排开的水量
  • 温度影响：不同温度下液体密度不同，未控制温度会导致密度测量偏差

综合实验 COMPREHENSIVE TEST — 三道综合测试题
TEST 01 · 计算题
一个体积为 2×10⁻³ m³ 的金属球，完全浸没在水中。（ρ水 = 1.0×10³ kg/m³，g = 10 N/kg）

(1) 求金属球受到的浮力大小。
(2) 若该金属球重 25 N，松手后金属球将如何运动？最终状态是什么？
(3) 求最终状态下金属球受到的浮力。
【参考答案】
(1) F浮 = ρ水gV排 = 1.0×10³ × 10 × 2×10⁻³ = 20 N
(2) F浮 (20N) < G (25N)，松手后下沉，最终沉底
(3) 沉底时 V排 = V物，故 F浮 = 20 N（不变）
TEST 02 · 实验分析题
小明用弹簧测力计测量某物体在空气中的重力为 8 N。他将物体缓慢浸入水中，测力计示数逐渐减小。当物体完全浸没时，测力计示数为 5 N。

(1) 求物体完全浸没时受到的浮力。
(2) 若将物体继续向下移动（仍完全浸没），测力计示数是否变化？为什么？
(3) 求物体的体积。（g = 10 N/kg）
(4) 求物体的密度。
【参考答案】
(1) F浮 = G − F′ = 8 − 5 = 3 N
(2) 不变。完全浸没后 V排 = V物 恒定，ρ液 和 g 也不变，由 F浮 = ρ液gV排 知浮力不变
(3) V物 = V排 = F浮 / (ρ水g) = 3 / (10³×10) = 3×10⁻⁴ m³
(4) m = G/g = 8/10 = 0.8 kg；ρ物 = m/V = 0.8 / (3×10⁻⁴) ≈ 2.67×10³ kg/m³
TEST 03 · 综合应用题
一艘轮船的排水量为 5000 t，从海水（ρ = 1.03×10³ kg/m³）驶入河水（ρ = 1.0×10³ kg/m³）。

(1) 求轮船受到的浮力。（g = 10 N/kg）
(2) 轮船在河水中是上浮一些还是下沉一些？为什么？
(3) 若轮船从河中驶回海中，船身会怎么变化？
【参考答案】
(1) 轮船漂浮：F浮 = G排 = m排·g = 5×10⁶ × 10 = 5×10⁷ N
(2) 下沉一些。浮力不变（始终等于船重），但河水密度较小，由 F浮 = ρ液gV排 可知，ρ液 减小则 V排 增大，因此船身下沉
(3) 上浮一些。海水密度更大，V排 减小，船身上浮
