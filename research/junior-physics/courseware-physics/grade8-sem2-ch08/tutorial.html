实验简报 LAB BRIEFING — 如何用一根棍子撬动世界？
⚙️ 引入问题

古埃及人建造金字塔时，需要将重达数吨的石块运送到高处。没有现代起重机，他们依靠的是杠杆和滑轮组等简单机械。

阿基米德曾说："给我一个支点，我就能撬动整个地球。"这句话虽然夸张，却道出了简单机械的核心思想——用较小的力完成较大的工作。

问题：杠杆真的能省力吗？省力的代价是什么？滑轮组最多能省多少力？这些机械真的"创造了"能量吗？

🔧 本节实验目标
  1. 认识杠杆的五要素，探究杠杆的平衡条件
  2. 理解定滑轮、动滑轮、滑轮组的省力原理
  3. 掌握功的定义和做功的两个必要因素
  4. 理解功率的物理意义及功率与速度的关系
  5. 认识动能与势能，理解机械能守恒的条件
核心思考题：一位工人用滑轮组将货物提升到三楼。货物越重，他做的功越多，但使用滑轮组后他用的力反而更小了——这"省"下来的力，到底去哪儿了？

📌 例题 N：一根杠杆的支点在一端，阻力作用在另一端，动力作用在杠杆中点。若阻力为 200 N，阻力臂为 1 m，动力臂为 2 m，求动力大小。

F₁ × 2 = 200 × 1

✏️ 例题 2

📌 例题 N：用动滑轮匀速提升重 400 N 的物体，不计滑轮重和摩擦，求拉力大小。若实际拉力为 250 N，求动滑轮的重力。
解：(1) 理想情况：动滑轮省一半力，F = G/2 = 400/2 = 200 N ✅
(2) 实际情况：F = (G + G动)/2
250 = (400 + G动)/2，G动 = 500 - 400 = 100 N ✅
动滑轮省力原理：理想 F = G/n（n为承担重物的绳子段数）

✏️ 例题 3

📌 例题 N：用滑轮组提升重 900 N 的物体，滑轮组由 3 段绳子承担重物，动滑轮总重 60 N。不计摩擦，求：(1) 拉力；(2) 若物体上升 2 m，绳子自由端移动多少？
解：(1) F = (G + G动)/n = (900 + 60)/3 = 960/3 = 320 N ✅
(2) s = n × h = 3 × 2 = 6 m ✅
n段绳子：F = (G+G动)/n，s = n×h（省力费距离）

杠杆 LEVER — 五要素 · 平衡条件 · 分类
📐 杠杆的定义与五要素

杠杆（Lever）：在力的作用下能绕固定点转动的硬棒。杠杆可以是直的，也可以是弯的。

五要素：

  • 支点（O）：杠杆绕其转动的固定点
  • 动力（F₁）：使杠杆转动的力
  • 阻力（F₂）：阻碍杠杆转动的力
  • 动力臂（L₁）：从支点到动力作用线的垂直距离
  • 阻力臂（L₂）：从支点到阻力作用线的垂直距离

杠杆五要素图示：支点O、动力F₁、阻力F₂、动力臂L₁、阻力臂L₂

⚖️ 杠杆的平衡条件

当杠杆处于静止或匀速转动状态时，杠杆平衡。通过大量实验可以发现：

F1 · L1 = F2 · L2

文字表述：动力 × 动力臂 = 阻力 × 阻力臂

这是杠杆的核心规律，也是后面分析所有杠杆问题的基础。

核心实验
探究杠杆的平衡条件

器材：杠杆（带刻度）、支架、钩码若干、弹簧测力计

实验步骤：

  1. 调节杠杆两端的平衡螺母，使杠杆在水平位置平衡（消除杠杆自重影响）
  2. 在杠杆左侧某位置挂若干钩码作为阻力 F₂，记录阻力臂 L₂

  4. 记录动力 F₁ 和动力臂 L₁

⚠ 误差分析 · 杠杆平衡实验
  • 杠杆自重：实验中假设杠杆自重可忽略或重心恰在支点处。若杠杆粗细不均，自重会产生额外力矩，导致 F₁L₁ ≠ F₂L₂
  • 平衡螺母调节：初始未调至水平平衡，杠杆自重产生的力矩会系统性地偏大或偏小一侧的乘积
  • 力的方向：若弹簧测力计拉力方向不与杠杆垂直，实际力臂小于读数对应的臂长，造成 F₁L₁ 计算偏大
  • 钩码质量误差：标称50g的钩码实际质量可能有±2g偏差，累积后影响数据规律性
  • 悬挂位置：钩码未挂在刻度线正上方，重心偏移导致力臂测量值不准确
🔧 杠杆的分类

根据动力臂与阻力臂的大小关系，杠杆可分为三类：

省力杠杆
L₁ > L₂
F₁ < F₂
撬棍、羊角锤、钢丝钳
省力但费距离
费力杠杆
L₁ < L₂
F₁ > F₂
镊子、筷子、钓鱼竿
费力但省距离
等臂杠杆
L₁ = L₂
F₁ = F₂
天平、定滑轮
不省力也不费力
核心规律：杠杆省力必然费距离，费力必然省距离。没有既省力又省距离的杠杆——这是功的原理决定的，能量守恒不允许"白嫖"。

滑轮 PULLEY SYSTEMS — 定滑轮 · 动滑轮 · 滑轮组
定滑轮

等臂杠杆

不省力，不省距离
改变力的方向

动滑轮

省力杠杆（L₁=2L₂）

省一半力，费一倍距离
不改变力的方向

滑轮组

既省力又改变方向

F = G/n，s = n·h
n = 承重绳段数

🔗 滑轮组省力判断方法

判断滑轮组省力情况的关键是数出承重绳段数 n（即直接承担动滑轮和重物总重的绳子段数）。

F = G总 / n　　s = n · h

G总 = G物 + G动（物重+动滑轮重）

n = 承重绳段数

s = 绳端移动距离

h = 物体上升高度

数 n 的技巧：只看动滑轮上连接的绳段数（包括从动滑轮引出的自由端）。定滑轮上的绳段不承重。

绕线口诀："奇动偶定"——承重段数 n 为奇数时绳端从动滑轮绕出，偶数时从定滑轮绕出

探究实验
测量滑轮组的机械效率

器材：滑轮组（1定1动）、弹簧测力计、钩码、刻度尺、铁架台

  1. 组装滑轮组，记录承重绳段数 n
  2. 用弹簧测力计测出钩码重力 G
  3. 缓慢匀速拉动测力计，记录拉力 F、物体上升高度 h、绳端移动距离 s
  4. 计算：有用功 W有 = G·h；总功 W总 = F·s；机械效率 η = W有/W总

⚠ 误差分析 · 滑轮组实验
  • 滑轮摩擦：滑轮轴承存在摩擦阻力，需要额外的拉力克服，导致实际拉力 F 大于理论值 G/n
  • 动滑轮自重：实验公式 F = G总/n 中，G总 包含动滑轮自重。若忽略动滑轮重，计算的"省力倍数"会偏大
  • 非匀速拉动：加速拉动时测力计示数偏大，减速时偏小。只有匀速拉动时读数才等于实际拉力
  • 绳重与绳摩擦：绳子自身有重量，且与滑轮、绳间存在摩擦，消耗额外能量
  • 读数视差：测力计未保持竖直或视线未与刻度盘垂直，导致读数误差

功 WORK — 定义 · 必要因素 · 计算
📐 功的定义

如果一个力作用在物体上，且物体在这个力的方向上移动了一段距离，就说这个力对物体做了功。

W = F · s

W — 功，单位：焦耳（J）

F — 作用在物体上的力（N）

s — 物体在力的方向上移动的距离（m）

1 J = 1 N·m

✅ 做功的两个必要因素

缺一不可：

  • ① 有力作用在物体上（F ≠ 0）
  • ② 物体在力的方向上移动了距离（s ≠ 0，且 s 与 F 同向）

四种典型情景：做功 / 有力无距离 / 力与位移垂直 / 靠惯性运动

📊 常见做功判断实例
情景 力 距离 是否做功 原因
推车前进 推力 F 前进 s ✓ 做功 F 与 s 同向
搬书静止不动 支持力 F s = 0 ✗ 不做功 无位移
提水桶水平行走 提力 F（竖直） 水平移动 s ✗ 不做功 F ⊥ s
踢出足球后飞行 脚已离开球 球飞行 s ✗ 不做功 靠惯性，无作用力
重力使苹果下落 重力 G 下落 h ✓ 做功 G 与 h 同向
记忆口诀："功由力和距离，方向要一致。有力无距不劳功，有距无力靠惯性，力距垂直白费劲。"

功率 POWER — 做功快慢 · 单位换算 · P=Fv
⚡ 功率的定义

功率（Power）：单位时间内所做的功，表示做功的快慢。

P = W / t

P — 功率，单位：瓦特（W）

W — 功，单位：焦耳（J）

t — 时间，单位：秒（s）

1 W = 1 J/s　　1 kW = 1000 W

🚗 功率与速度的关系

当物体在力 F 作用下以速度 v 匀速运动时：

P = F · v

推导：P = W/t = (F·s)/t = F·(s/t) = F·v

此公式在分析汽车上坡换挡、机车牵引等问题时非常有用。发动机功率一定时，速度越小，牵引力越大。

估测人上楼时的功率

原理：P = W/t = mgh/t

  1. 用体重秤测出自己的质量 m（kg）
  2. 数出从一楼到三楼的台阶数，计算上升高度 h（每层约 3m）
  3. 用秒表记录匀速上楼所用时间 t
  4. 计算功率 P = mgh/t

实验员质量 m = 　　　　 kg

上升高度 h = 　　　　 m（如：2层 × 3m = 6m）

上楼时间 t = 　　　　 s

克服重力做功 W = mgh = 　　　　 J

功率 P = W/t = 　　　　 W ≈ 　　　　 W

与家用电器对比：约等于 　　　　 W 的电灯泡功率

✏️ 例题 1

📌 例题 N：一根杠杆的支点在一端，阻力作用在另一端，动力作用在杠杆中点。若阻力为 200 N，阻力臂为 1 m，动力臂为 2 m，求动力大小。
解：杠杆平衡条件：F₁ × L₁ = F₂ × L₂
F₁ × 2 = 200 × 1
F₁ = 200/2 = 100 N ✅
杠杆平衡：动力 × 动力臂 = 阻力 × 阻力臂

✏️ 例题 2

📌 例题 N：用动滑轮匀速提升重 400 N 的物体，不计滑轮重和摩擦，求拉力大小。若实际拉力为 250 N，求动滑轮的重力。
解：(1) 理想情况：动滑轮省一半力，F = G/2 = 400/2 = 200 N ✅
(2) 实际情况：F = (G + G动)/2
250 = (400 + G动)/2，G动 = 500 - 400 = 100 N ✅
动滑轮省力原理：理想 F = G/n（n为承担重物的绳子段数）

✏️ 例题 3

📌 例题 N：用滑轮组提升重 900 N 的物体，滑轮组由 3 段绳子承担重物，动滑轮总重 60 N。不计摩擦，求：(1) 拉力；(2) 若物体上升 2 m，绳子自由端移动多少？
解：(1) F = (G + G动)/n = (900 + 60)/3 = 960/3 = 320 N ✅
(2) s = n × h = 3 × 2 = 6 m ✅
n段绳子：F = (G+G动)/n，s = n×h（省力费距离）
