🔦 01 光的直线传播

光在同种均匀介质中沿直线传播。这是光学实验室行动的第一法则——路径最短，速度最快。

光速是宇宙极限速度：
c = 3 × 10⁸ m/s = 300,000 km/s
光从太阳到地球只需约 8 分钟。

📐 影子形成原理

当光遇到不透明物体时，被阻挡的区域形成影子。影子的大小与光源位置和物体位置有关。

45°

拖动滑块改变太阳角度，观察影子长度变化

🌑 日食与月食

日食：月球运行到太阳和地球之间，月球的影子落在地球上。
月食：地球运行到太阳和月球之间，地球的影子落在月球上。

✏️ 例题 1

一棵树高 3m，太阳光与地面成 30° 角，求树影长度。

解：tan 30° = 树高 / 影长 → 影长 = 3 / tan 30° ≈ 3 / 0.577 ≈ 5.2 m

🪞 02 光的反射

光遇到物体表面时会反射。光学实验室利用反射定律完成"镜面弹跳"战术。

反射定律
入射角 i = 反射角 r
入射光线、反射光线与法线在同一平面内

🔄 镜面反射 vs 漫反射

🖼️ 平面镜成像

平面镜成像特点：
① 像与物大小相等
② 像到镜面距离 = 物到镜面距离
③ 成虚像（光线反向延长线相交）
④ 像与物关于镜面对称
150px

✏️ 例题 2

一个人站在平面镜前 2m 处，他的像距离他多远？若他向镜面走近 1m，像距变为多少？

解：初始像距人 = 2 + 2 = 4 m。走近1m后，物距=1m，像距人 = 1 + 1 = 2 m。

💎 03 光的折射

光从一种介质进入另一种介质时，传播方向发生偏折，这就是折射。

折射规律
光从空气 → 水/玻璃：折射角 < 入射角（靠近法线）
光从水/玻璃 → 空气：折射角 > 入射角（远离法线）

🌊 折射演示

30°

🌈 光的色散

白光通过三棱镜时，被分解成七种颜色的光——光谱。

色散顺序（折射率从小到大）：
红 橙 黄 绿 蓝 靛 紫
红光偏折最小，紫光偏折最大。

牛顿三棱镜实验：白光进入，七彩光出射

✏️ 例题 3

光从空气射入水中，入射角为 45°，折射角为 32°。若入射角增大到 60°，折射角会如何变化？

解：入射角增大，折射角也增大，但始终保持折射角 < 入射角（光从空气进入水时）。

🔍 04 透镜与成像

透镜是光学实验室的核心装备，掌握凸透镜成像规律是完成任务的关键。

📊 凸透镜成像规律

物距 u 像距 v 像的性质 应用
u > 2f f < v < 2f 倒立、缩小、实像 📷 照相机
u = 2f v = 2f 倒立、等大、实像 📏 测焦距
f < u < 2f v > 2f 倒立、放大、实像 🎬 投影仪
u < f — 正立、放大、虚像 🔍 放大镜
2.5f

拖动滑块改变物距，观察成像变化（设 f = 60px）

✏️ 例题 4

用焦距为 10cm 的凸透镜做实验，当蜡烛距离透镜 25cm 时，光屏应放在什么位置才能得到清晰的像？像是放大还是缩小？

解：u = 25cm > 2f (20cm)，所以成倒立缩小实像，像距 f < v < 2f，即 10cm < v < 20cm。像是缩小的。

📋 05 公式与要点总结

⚡ 光速：c = 3 × 10⁸ m/s（真空中）

🪞 反射定律：∠i = ∠r（入射角 = 反射角）

💎 折射规律：空气→水/玻璃，折射角 < 入射角

🔍 凸透镜成像：u > 2f → 缩小实像；f < u < 2f → 放大实像；u < f → 放大虚像
