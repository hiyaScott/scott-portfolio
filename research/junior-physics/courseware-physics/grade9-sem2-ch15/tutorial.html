02

电磁波的产生 Generation · 从振荡到辐射

麦克斯韦的预言

1865年，麦克斯韦建立电磁场理论，预言了电磁波的存在：

  • 变化的电场产生磁场
  • 变化的磁场产生电场
  • 周期性变化的电磁场在空间传播，形成电磁波

1888年，赫兹通过实验首次证实电磁波的存在，为人类无线通信开辟了道路。

LC振荡电路

由电感线圈 L 和电容器 C 组成的回路，可以产生高频电磁振荡，向外辐射电磁波。

L — 电感线圈
C — 电容器
∿∿∿∿∿
电磁波辐射
频率 f = 1 / (2π√LC)
振荡周期 T = 2π√(LC)   |   频率 f = 1/T = 1 / (2π√LC)
  • 电感 L 越大 → 振荡频率越低
  • 电容 C 越大 → 振荡频率越低
  • 实际应用中通过调节 L 或 C 来改变发射频率（调谐）

波速公式 c = λf

c = λ · f
c = 波速（真空中约 3×10⁸ m/s）  λ = 波长（m）  f = 频率（Hz）

在真空中，所有电磁波的传播速度都等于光速 c。波长与频率成反比关系。

03

电磁波的传播 Propagation · 穿越空间

不需要介质

电磁波是电磁场本身的传播，不需要任何介质。这是它与机械波（声波、水波）最根本的区别。

  • 声波传播需要空气、水、固体等物质介质
  • 电磁波在真空中可以自由传播（如太阳光穿越太空到达地球）
  • 电磁波也可以在介质中传播，但速度会降低

真空中的光速

c = 299,792,458 m/s ≈ 3×10⁸ m/s
真空中的光速是宇宙中的基本物理常数，也是电磁波传播的速度上限
  • 太阳光到达地球约需 8 分 20 秒（距离约 1.5×10¹¹ m）
  • 月球反射的光到达地球约需 1.28 秒
  • 电磁波绕地球赤道一周约需 0.13 秒

不同介质中的传播速度

电磁波进入介质后，传播速度降低，折射率 n = c/v。

介质 折射率 n 波速 v = c/n 变化
真空1.0003.00×10⁸ m/s基准
空气1.0003≈ 2.999×10⁸ m/s几乎不变
水1.332.26×10⁸ m/s降低 25%
玻璃1.5 ~ 1.91.58 ~ 2.00×10⁸ m/s降低 33~47%
钻石2.421.24×10⁸ m/s降低 59%

注意：电磁波在介质中的速度降低，但频率保持不变，因此波长也会相应缩短（λ' = λ/n）。

电磁波的应用 Applications · 从广播到医疗

不同电磁波应用场景对照

电磁波 频率/波长 典型应用 工作原理简述
长波无线电 30~300 kHz 潜艇通信、导航 绕射能力强，可穿透海水
中短波广播 300 kHz~30 MHz AM/FM 广播电台 调幅/调频调制载波传输音频
微波 300 MHz~300 GHz 雷达、微波炉、卫星通信 雷达：反射测距；微波炉：水分子共振发热
红外线 300 GHz~400 THz 遥控器、热成像、理疗 热效应显著，用于感知温度
可见光 400~790 THz 照明、光纤通信、摄影 人眼可感知的电磁波段
紫外线 750 THz~30 PHz 消毒杀菌、验钞、促进VD合成 光子能量足以破坏微生物DNA
X射线 30 PHz~30 EHz 医学影像、安检、工业探伤 穿透软组织，被骨骼吸收成像
γ射线 > 30 EHz 肿瘤放疗、食品灭菌、天文观测 高能光子，可破坏细胞DNA

重点应用详解

▍无线电广播

广播电台通过调制将音频信号加载到高频载波上。AM（调幅）改变载波的振幅，FM（调幅）改变载波的频率。收音机通过调谐电路选择特定频率，经解调还原出声音信号。

▍电视信号传输

电视信号包含图像信号和伴音信号。图像信号通过调幅方式调制到甚高频（VHF）或超高频（UHF）载波上，伴音信号通过调频方式调制。数字电视采用MPEG编码压缩后传输。

▍移动通信（手机）

手机同时是发射机和接收机。通话时，手机将声音信号数字化后调制到微波频段，通过基站天线发射；接收时逆向执行该过程。蜂窝网络通过频率复用实现广域覆盖。

▍雷达（RADAR）

RAdio Detection And Ranging。发射微波脉冲，接收目标反射的回波，通过时间差计算距离（s = c·t/2），通过多普勒频移计算速度。广泛应用于航空管制、气象预报、自动驾驶。

▍微波炉

磁控管产生频率约2.45 GHz的微波。食物中的水分子是极性分子，在交变电场作用下高速旋转摩擦生热。微波穿透深度约几厘米，大块食物需热传导加热内部。

▍X射线成像

X射线穿透人体软组织时衰减较小，遇到骨骼等高Z材料时显著吸收。在胶片或数字探测器上形成明暗对比影像。CT扫描通过多角度投影重建三维图像。

信息的传递 Transmission · 信号与信道

模拟信号 vs 数字信号

模拟信号 Analog

  • 信号连续变化，取值无限
  • 易受噪声干扰，失真累积
  • 传统电话、广播使用
  • 带宽利用率相对较低

数字信号 Digital

  • 仅取 0 或 1 两种状态
  • 抗干扰强，可纠错编码
  • 现代通信、计算机使用
  • 便于加密、压缩、存储

现代通信系统普遍采用数字信号。模拟信号在发送端通过ADC转换为数字信号，接收端通过DAC还原。

光纤通信

光纤利用全反射原理将光信号（激光）约束在玻璃纤维中传播，实现高速、大容量、低损耗的信息传输。

激光输入 →
包层（折射率低） 纤芯（折射率高） 包层（折射率低）
→ 光信号输出

全反射条件：光从光密介质射向光疏介质，入射角 ≥ 临界角 θc = arcsin(n₂/n₁)

  • 传输速率：单纤可达 Tbps 级（每秒万亿比特）
  • 传输损耗：现代光纤每公里仅衰减约 0.2 dB
  • 不受电磁干扰：光纤传输的是光，不受雷电、电磁脉冲影响
  • 保密性好：光信号不向外辐射，难以窃听
  • 海底光缆：连接各大洲的互联网骨干网络

卫星通信

利用人造地球卫星作为中继站转发无线电波，实现地球站之间的远距离通信。

  • 同步轨道卫星（GEO）：高度约 35,786 km，相对地面静止，3颗即可覆盖全球（除两极）
  • 中轨道卫星（MEO）：高度 2,000~20,000 km，如GPS卫星（约20,200 km）
  • 低轨道卫星（LEO）：高度 160~2,000 km，延迟低，需大量卫星组网（如Starlink）
  • 上行链路（地面→卫星）与下行链路（卫星→地面）使用不同频段避免干扰
  • 卫星通信覆盖广，不受地形限制，是偏远地区通信的重要手段

5G 通信技术

  • 增强移动宽带（eMBB）：峰值下载速率可达 20 Gbps，支持 4K/8K 视频、VR/AR 应用
  • 超可靠低延迟（uRLLC）：延迟降至 1 ms 级别，支撑远程手术、自动驾驶
  • 海量机器通信（mMTC）：每平方公里可连接百万级设备，物联网基础
  • 使用 Massive MIMO（大规模天线阵列）和波束赋形技术提升频谱效率
  • 毫米波（mmWave，24 GHz以上）提供超大带宽，但传播距离短、穿透力弱

WiFi 与蓝牙

两种最常见的短距离无线通信技术，工作于 2.4 GHz ISM 免授权频段。

特性 WiFi (802.11) 蓝牙 (Bluetooth)

传输距离30~100 m10~100 m（Class 1/2/3）
数据速率最高 9.6 Gbps (WiFi 7)最高 2 Mbps (BLE 5.0)

GPS / 北斗卫星导航

全球导航卫星系统（GNSS）通过接收多颗卫星发射的无线电信号，利用三角定位原理计算接收器的精确位置。

  • GPS（美国）：24颗中轨道卫星，民用信号 L1: 1575.42 MHz，定位精度约 3~5 m
  • 北斗（中国）：混合轨道星座（GEO + IGSO + MEO），亚太地区精度优于 1 m
  • 伽利略（欧盟）、GLONASS（俄罗斯）：多系统联合定位可进一步提升精度
  • 定位原理：接收器测量来自至少4颗卫星的信号传播时间，解算三维坐标（x, y, z）和时间

综合实验 Comprehensive Lab · 3道综合测试题

TEST 01 / 概念辨析
关于电磁波，下列说法正确的是（　　）
  • 电磁波不能在真空中传播
  • 电磁波的传播速度与介质无关，恒等于光速
  • 振荡电路中的电容器放电完毕时，电路中的电流达到最大值
  • 调幅广播（AM）比调频广播（FM）抗干扰能力更强
答案：C
解析：A 错——电磁波可以在真空中传播；B 错——电磁波在介质中速度降低（v = c/n）；C 对——电容器放电完毕时电场能全部转化为磁场能，电流最大；D 错——FM 抗干扰能力远强于 AM。
TEST 02 / 计算应用
某广播电台发射的电磁波波长为 600 m，求：
（1）该电磁波的频率是多少？
（2）若某收音机的 LC 调谐电路中电感 L = 0.5 mH，要接收到该电台信号，电容器的电容 C 应调至多大？
（已知真空中光速 c = 3×10⁸ m/s）
答案：
（1）由 c = λf 得：f = c/λ = 3×10⁸ / 600 = 5×10⁵ Hz = 500 kHz

（2）由 LC 振荡频率公式 f = 1 / (2π√LC) 变形得：
C = 1 / (4π²f²L) = 1 / (4π² × (5×10⁵)² × 0.5×10⁻³)
= 1 / (4 × 3.14² × 25×10¹⁰ × 0.5×10⁻³)
≈ 2.03×10⁻¹⁰ F = 203 pF

结论：该电台频率为 500 kHz，收音机需将电容调至约 203 pF。
TEST 03 / 综合探究
阅读材料，回答问题：

"2024年，中国北斗三号全球卫星导航系统全面建成开通，向全球用户提供定位导航授时服务。北斗系统由 30 颗卫星组成，包括 3 颗地球静止轨道卫星、3 颗倾斜地球同步轨道卫星和 24 颗中圆地球轨道卫星。卫星向地面发射频率为 1561.098 MHz（B1频段）和 1207.140 MHz（B2频段）的无线电信号。"

（1）北斗卫星向地面传输信号利用的是什么波？这种波的传播需要介质吗？
（2）计算 B1 频段信号的波长（保留整数）。
（3）为什么卫星导航系统至少需要 4 颗卫星才能确定接收器的位置？
（4）北斗采用三种轨道混合组网，相比单一 GPS 中轨道方案有什么优势？
答案：
（1）电磁波（微波/无线电波）；不需要介质，可在真空中传播。

（2）λ = c/f = 3×10⁸ / (1561.098×10⁶) ≈ 0.192 m ≈ 19 cm（约分米波）。

（3）需要解算 4 个未知数：接收器的三维坐标（x, y, z）和接收机钟差。每颗卫星提供一个距离方程，4 颗卫星建立 4 个方程，才能解出 4 个未知数。

（4）混合轨道优势：GEO 卫星对亚太高仰角覆盖，抗遮挡能力强；IGSO 卫星增强区域覆盖；MEO 卫星实现全球覆盖。亚太地区可见卫星数多于 GPS，定位精度更高、可靠性更强。
