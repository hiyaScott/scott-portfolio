磁现象
Module MAG-02 · 磁性 · 磁极 · 磁化 · 去磁
磁性

物体吸引铁、钴、镍等物质的性质叫做磁性。具有磁性的物体称为磁体。

磁体按来源可分为天然磁体（磁铁矿）和人造磁体（钢条、铁氧体）。按磁性保持时间可分为永磁体和软磁体。

磁极

磁体上磁性最强的部分叫做磁极。每个磁体都有两个磁极：

  • 南极（S极）：指向地理南极的磁极
  • 北极（N极）：指向地理北极的磁极

磁极间相互作用规律：同名磁极相斥，异名磁极相吸。

磁化与去磁

磁化

使原来没有磁性的物体获得磁性的过程。方法包括：

  • 与磁体接触或靠近
  • 用磁体沿同一方向摩擦
  • 通入电流（电磁铁原理）

去磁

使磁体失去磁性的过程。方法包括：

  • 剧烈敲击或加热
  • 通入反向交变电流
  • 将磁体置于交流线圈中逐渐移出
磁性材料分类：硬磁材料（永磁体）┋ 软磁材料（易磁化也易去磁）┋ 铁氧体

磁场
Module MAG-03 · 磁感线 · 磁场方向 · 地磁场 · 磁偏角
磁场

磁场是磁体周围存在的一种特殊物质，它对放入其中的磁体产生力的作用。磁场具有方向性和强弱的分布特征。

磁场的基本性质：对放入其中的磁体产生磁力作用。磁场看不见摸不着，但可以通过它对其他磁体的作用来感知。

磁感线

为了形象地描述磁场，引入磁感线这一假想曲线：

  • 磁感线上任意一点的切线方向表示该点的磁场方向
  • 磁感线的疏密程度表示磁场的强弱
  • 在磁体外部，磁感线从N极出发回到S极
  • 在磁体内部，磁感线从S极回到N极
  • 磁感线永不相交，是闭合曲线
磁场方向

磁场中某点的方向规定为：在该点可以自由转动的小磁针静止时北极所指的方向。

或者说：磁场中某点的磁场方向就是该点磁感线的切线方向。

磁场方向 ≡ 小磁针N极受力方向 ≡ 磁感线切线方向

地磁场

地球本身就是一个巨大的磁体，周围存在地磁场。地磁场的特性：

  • 地磁北极在地理南极附近
  • 地磁南极在地理北极附近
  • 地磁场磁感线从地理南极附近发出，回到地理北极附近
  • 指南针N极指向地理北方，实际是地磁南极的吸引
磁偏角

地磁两极与地理两极并不重合，它们之间存在一定的夹角，称为磁偏角。

中国宋代科学家沈括在《梦溪笔谈》中最早记载了磁偏角现象，比西方早了四百多年。

磁偏角 = 磁子午线与地理子午线的夹角

磁偏角随地理位置和时间变化，精确导航需要实时校正。

电流的磁场
Module MAG-04 · 奥斯特实验 · 安培定则 · 电磁铁
奥斯特实验 1820 · Hans Christian Ørsted

实验现象：将通电导线平行放置在小磁针上方，小磁针会发生偏转；切断电流后，小磁针恢复原来的指向。

实验结论：通电导线周围存在磁场，即电流具有磁效应。这是人类第一次发现电与磁之间的联系。

MAG-FIG-03 · 奥斯特实验：电流方向→时，磁针N极向纸内偏转（垂直纸面向里）
安培定则（右手螺旋定则）

用于判断通电直导线周围磁场的方向：

右手握住导线，拇指指向电流方向
四指弯曲方向 = 磁感线环绕方向

通电直导线周围磁感线是围绕导线的同心圆，距离导线越远，磁场越弱。

安培定则（螺线管）

用于判断通电螺线管的磁极方向：

右手握住螺线管，四指指向电流方向
拇指指向 = 螺线管N极方向

通电螺线管外部的磁感线分布与条形磁体相似，它也有两个磁极。

MAG-FIG-04 · 安培定则手势图解：直导线 vs 螺线管
电磁铁

电磁铁是带有铁芯的通电螺线管。铁芯被磁化后大大增强了螺线管的磁性。电磁铁相比永磁体具有显著优势：

  • 磁性的有无可以通过通断电来控制
  • 磁性的强弱可以通过电流大小和线圈匝数来调节
  • 磁极的方向可以通过改变电流方向来控制

影响电磁铁磁性强弱的因素实验：

影响因素 控制变量 实验操作 现象与结论
线圈匝数 电流大小、铁芯相同 改变线圈匝数，观察吸引大头针数量 匝数越多，磁性越强
电流大小 线圈匝数、铁芯相同 调节滑动变阻器改变电流 电流越大，磁性越强
有无铁芯 线圈匝数、电流相同 对比有铁芯与无铁芯情况 有铁芯时磁性显著增强
电磁铁磁性影响因素：匝数 N ↑ → 磁性 ↑ ┋ 电流 I ↑ → 磁性 ↑ ┋ 铁芯存在 → 磁性显著 ↑

磁场对电流的作用
Module MAG-05 · 电动机原理 · 左手定则 · 换向器
磁场对通电导体的作用力

将通电导体放入磁场中，导体会受到磁场力的作用而发生运动。这是将电能转化为机械能的基础原理。

受力方向与电流方向和磁场方向都有关。改变电流方向或磁场方向，导体受力方向都会改变；若两者同时改变，受力方向不变。

左手定则

用于判断通电导体在磁场中受力方向：

伸开左手，使拇指与其余四指垂直，且在同一平面
磁感线垂直穿入手心（B → 手心）
四指指向电流方向（I → 四指）
拇指指向导体受力方向（F → 拇指）
记忆口诀：B-I-F 手心-四指-拇指
MAG-FIG-05 · 磁场对通电导体的作用力 — 电动机工作原理
电动机原理

直流电动机是根据通电线圈在磁场中受力转动的原理制成的。

工作过程：通电线圈在磁场中受到力的作用而转动，将电能转化为机械能。

电动机的转动方向取决于电流方向和磁场方向。要改变转向，只需改变其中一个方向。

换向器

当线圈转过平衡位置时，换向器自动改变线圈中的电流方向，使线圈能够持续向同一方向转动。

换向器由两个半铜环组成，与线圈两端相连，随线圈一起转动。电刷固定不动，分别与两个半环接触。

无换向器 → 线圈在平衡位置来回摆动
有换向器 → 线圈持续单向转动

电磁感应
Module MAG-06 · 法拉第实验 · 感应电流 · 发电机 · 右手定则
法拉第实验 1831 · Michael Faraday

实验装置：将导体棒放入磁场中，两端连接灵敏电流计，形成闭合回路。

实验现象：

  • 导体静止在磁场中 → 电流计指针不偏转
  • 导体切割磁感线运动 → 电流计指针偏转（有感应电流）
  • 导体运动方向相反 → 指针偏转方向相反
  • 导体运动速度越快 → 指针偏转幅度越大

实验结论：闭合电路的一部分导体在磁场中做切割磁感线运动时，导体中会产生感应电流。这种现象叫做电磁感应。

MAG-FIG-06 · 发电机原理图解：机械能 → 切割磁感线 → 感应电流（电能）
感应电流产生的条件

产生感应电流必须同时满足三个条件：

  • 闭合电路 — 电路必须是通路
  • 一部分导体 — 只有部分导体在磁场中运动
  • 切割磁感线 — 导体的运动方向与磁感线不平行
感应电流大小 ∝ 磁场强度 ┋ 导体运动速度 ┋ 导体有效长度
发电机原理

交流发电机是根据电磁感应原理制成的。线圈在磁场中转动，切割磁感线，产生感应电流。

工作过程：将机械能转化为电能。线圈每转过半圈（经过平衡位置），感应电流方向改变一次。

中国电网交流电频率：50Hz，即线圈每秒转动50圈，电流方向每秒改变100次。

右手定则（感应电流方向）

用于判断导体切割磁感线时感应电流的方向：

伸开右手，使拇指与其余四指垂直，且在同一平面
磁感线垂直穿入手心（B → 手心）
拇指指向导体运动方向（v → 拇指）
四指指向感应电流方向（I → 四指）
记忆口诀：B-v-I 手心-拇指-四指
左手定则 vs 右手定则
对比项 左手定则 右手定则
适用场景 通电导体在磁场中受力运动 导体切割磁感线产生感应电流
能量转化 电能 → 机械能（电动机） 机械能 → 电能（发电机）
因果关系 因：有电流 → 果：受力 因：运动 → 果：生电
拇指 受力方向 (F) 运动方向 (v)
四指 电流方向 (I) 感应电流方向 (I)
手心 磁感线垂直穿入 (B)
记忆法 「力」字向左撇 → 左手 「电」字向右弯 → 右手

综合实验
Module MAG-07 · 3道综合测试题
Q1
电磁铁综合实验设计

某实验小组要探究电磁铁磁性强弱与线圈匝数、电流大小、有无铁芯的关系。现有以下器材：电源、滑动变阻器、电流表、开关、若干相同铁钉、漆包线、大头针、导线若干。

请回答：

(1) 实验中通过观察什么现象来判断电磁铁磁性强弱？

(2) 如何设计实验来分别探究上述三个因素的影响？（写出控制变量和实验步骤要点）

(3) 画出探究"磁性强弱与电流大小关系"的实验电路图。

[参考答案]

(1) 通过观察电磁铁吸引大头针的数量来判断磁性强弱。吸引越多，磁性越强。这种方法称为转换法。

(2) 探究各因素需控制变量：

  • 与匝数关系：控制电流大小、铁芯相同，改变线圈匝数，比较吸引大头针数量
  • 与电流关系：控制线圈匝数、铁芯相同，调节滑动变阻器改变电流，比较吸引大头针数量
  • 与铁芯关系：控制线圈匝数、电流相同，对比有铁芯和无铁芯（空心）情况

(3) 电路图：电源 → 开关 → 滑动变阻器 → 电流表 → 电磁铁线圈 → 回到电源负极（串联电路）。

Q2
电动机与发电机原理辨析

如图所示的装置，AB是放在磁场中的导体棒，与灵敏电流计和开关组成闭合回路。

请回答：

(1) 闭合开关后，若用手使导体AB水平向右运动，电流计指针会偏转吗？向哪边偏转？这是什么原理的应用？

(2) 若闭合开关后导体AB不动，用外力使蹄形磁铁水平向左运动，电流计指针会偏转吗？为什么？

(3) 若将图中的电流计换成电源，闭合开关后导体AB会发生什么现象？这是什么原理的应用？用什么定则判断导体运动方向？

[参考答案]

(1) 会偏转。导体AB向右运动切割磁感线，产生感应电流，电流计指针偏转。这是电磁感应原理的应用，是发电机的工作原理。

(2) 会偏转。因为运动是相对的——导体AB相对于磁铁向右运动，仍然切割磁感线，满足电磁感应条件，产生感应电流。

(3) 导体AB会受力运动。这是通电导体在磁场中受力的原理，是电动机的工作原理。用左手定则判断导体运动方向。

Q3
安培定则与左右手定则综合应用

如图所示，在通电螺线管的上方放置一个小磁针，右侧水平放置一根通电直导线。

请回答：

(1) 根据电源正负极判断螺线管的N、S极，并在图中标出。

(2) 小磁针静止时N极指向何方？请在图中标出小磁针的指向。

(3) 若通电直导线的电流方向垂直纸面向外（⊙），在螺线管右侧区域，直导线受到的磁场力方向是向上还是向下？请用左手定则说明判断过程。

(4) 若将通电直导线改为沿图示方向运动切割磁感线，用右手定则判断导线中感应电流的方向。

[参考答案]

(1) 用安培定则（螺线管）：右手四指沿电流方向握住螺线管，拇指指向即为N极。根据电源极性判断电流方向后，可确定螺线管左端为N极、右端为S极（或反之，视具体电路而定）。

(2) 小磁针N极指向螺线管的S极（异名磁极相吸）。若螺线管左端为N极、右端为S极，则小磁针N极指向右方（指向螺线管S极）。

(3) 判断步骤：① 先确定螺线管在右侧区域产生的磁场方向（由N极指向S极，在螺线管右侧磁场方向向右）；② 伸开左手，让磁感线穿入手心（手心朝左）；③ 四指指向电流方向（垂直纸面向外，即指向读者）；④ 拇指指向上方，即直导线受力方向向上。

(4) 用右手定则：磁感线穿入手心（方向向右，手心朝左），拇指指向运动方向，四指指向即为感应电流方向。根据具体运动方向可判断电流是垂直纸面向里还是向外。
