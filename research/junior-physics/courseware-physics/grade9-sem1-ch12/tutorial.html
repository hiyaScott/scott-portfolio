电阻 RESISTANCE

定义与单位

电阻表示导体对电流的阻碍作用。导体对电流的阻碍越大，电阻越大。

R = U / I
R = 电阻(Ω)    U = 电压(V)    I = 电流(A)
单位换算
1 MΩ = 106 Ω   |   1 kΩ = 103 Ω   |   1 Ω = 10-3 kΩ = 10-6 MΩ

电阻的决定因素

导体的电阻是导体本身的一种性质，它的大小取决于以下四个因素：

R = ρ · L / S
ρ = 电阻率(Ω·m)    L = 长度(m)    S = 横截面积(m²)
影响因素对照
  • 材料 — 不同材料的电阻率 ρ 不同，银 < 铜 < 铝 < 铁
  • 长度 — R ∝ L，导体越长电阻越大（正比关系）
  • 横截面积 — R ∝ 1/S，导体越粗电阻越小（反比关系）
  • 温度 — 金属温度越高电阻越大；半导体、电解液温度越高电阻越小

关键结论

导体的电阻 R 与长度 L 成正比，与横截面积 S 成反比，还与材料和温度有关。 公式中 ρ 为电阻率，单位 Ω·m，是材料的固有属性。

变阻器 RHEOSTAT / POTENTIOMETER

滑动变阻器原理

滑动变阻器通过改变接入电路中电阻丝的长度来改变电阻，从而改变电路中的电流和电压。 核心结构：电阻丝绕在瓷筒上，滑片可在金属杆上滑动，接线柱为「两下一上」或「一上一下」。

▲ 图1：滑动变阻器结构示意（滑片P位于中点位置）

「一上一下」接法

滑动变阻器必须采用「一上一下」的接线方式才能起到变阻作用。 即一个接线柱接金属杆（上），另一个接电阻丝端（下）。

✓ 有效接法

A-C 或 A-D（左上 + 任一下）
B-C 或 B-D（右上 + 任一下）

接入电路的电阻丝长度随滑片移动而改变，实现连续调阻。

✗ 无效接法

A-B（两上）→ 电阻≈0，相当于导线
C-D（两下）→ 电阻=max，相当于定值电阻

两上接法无法变阻；两下接法滑片移动无效，起不到调节作用。

▲ 图2：有效 vs 无效接法电路示意（两下接法电阻恒为最大值）
在电路中的作用
  • 限流 — 串联在电路中，通过调节自身阻值控制总电流大小
  • 分压 — 与负载串联构成分压电路，调节负载两端电压
  • 保护电路 — 闭合开关前置于最大阻值，防止电流过大损坏元件
3
欧姆定律 OHM'S LAW

核心内容

德国物理学家乔治·西蒙·欧姆（Georg Simon Ohm）于1827年提出： 导体中的电流，跟导体两端的电压成正比，跟导体的电阻成反比。

I = U / R
变形公式：
U = I · R      R = U / I
适用条件

1. 纯电阻电路（电能全部转化为内能）
2. 同一导体、同一时刻的 U、I、R
3. 温度不变（电阻恒定）
4. 金属导体、电解液导电适用

不适用情况

1. 含电动机、电感、电容的非纯电阻电路
2. 气体导电（日光灯、霓虹灯）
3. 温度变化剧烈的场合
4. 半导体二极管（非线性元件）

理解要点

R = U/I 是电阻的计算式，不是决定式。 导体的电阻由其材料、长度、横截面积和温度决定，与电压和电流无关。 即使不接入电路（U=0, I=0），导体的电阻依然存在。

U-I 图像特征

对于定值电阻，U-I 图像是过原点的直线；斜率即为电阻值。

串并联电阻 SERIES & PARALLEL RESISTANCE

串联电路

R串 = R1 + R2 + R3 + ...
串联总电阻 = 各电阻之和（总电阻大于任意分电阻）
串联分压原理

串联电路中，各电阻两端电压与其阻值成正比：

U1 / U2 = R1 / R2

阻值越大的电阻，分担的电压越多。此原理是电压表扩大量程的基础。

并联电路

1/R并 = 1/R1 + 1/R2 + 1/R3 + ...
并联总电阻的倒数 = 各电阻倒数之和（总电阻小于任意分电阻）
R并 = (R1 · R2) / (R1 + R2)
两电阻并联的快捷计算公式
并联分流原理

并联电路中，各支路电流与其阻值成反比：

I1 / I2 = R2 / R1

阻值越小的支路，流过的电流越大。此原理是电流表扩大量程的基础。

伏安法测电阻 VOLT-AMMETER METHOD

实验原理

用电压表测出待测电阻两端的电压 U，用电流表测出通过电阻的电流 I， 根据欧姆定律的变形公式 R = U/I 计算出电阻值。 为减小误差，通常测量多组数据取平均值。

误差分析：内接 vs 外接

外接法 · 误差来源

电压表分流 → 电流表读数 偏大
I测 = IR + IV

测量值 R测 = U/I测
< R真（测量值偏小）

适用：Rₓ ≪ Rᵥ（小电阻，Rₓ < √(Rₐ·Rᵥ)）

内接法 · 误差来源

电流表分压 → 电压表读数 偏大
U测 = UR + UA

测量值 R测 = U测/I
> R真（测量值偏大）

适用：Rₓ ≫ Rₐ（大电阻，Rₓ > √(Rₐ·Rᵥ)）

其他误差来源
  • 电表内阻 — 理想电表不存在，实际电表均有有限内阻
  • 接触电阻 — 接线柱、导线接头存在额外接触电阻
  • 温度变化 — 通电发热导致被测电阻阻值漂移
  • 读数误差 — 指针式电表的视差和估读误差
  • 电源波动 — 电池内阻随放电增大，输出电压不稳

电阻定律应用 — 导线改造问题

一根镍铬合金丝电阻为 20Ω，现将它均匀拉长至原长的 2 倍， 然后从拉长后的导线中截取一半使用。问：截取后这段导线的电阻是多少？

若将原导线对折后拧成一股使用，此时电阻又是多少？

◆ HINT 拉长过程体积 V = L·S 保持不变。拉长 2 倍 → 横截面积变为原来的 1/2。 对折后相当于两根并联，长度减半、横截面积加倍。
