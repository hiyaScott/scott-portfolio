02
电阻
RESISTANCE
R = U / I
单位：欧姆 (Ω) — 1Ω = 1V / 1A
定义与物理意义

电阻是导体对电流的阻碍作用。导体两端电压与通过电流的比值为一常量，该常量即导体的电阻。电阻是导体本身的属性，与电压、电流无关。

电阻定律 — 决定因素

R = ρ · L / S
ρ — 电阻率 (Ω·m)  |  L — 长度 (m)  |  S — 横截面积 (m²)
材料电阻率对照表 (20°C) DAT-01
材料电阻率 ρ (Ω·m)分类
银1.6 × 10⁻⁸导体
铜1.7 × 10⁻⁸导体
铝2.9 × 10⁻⁸导体
铁1.0 × 10⁻⁷导体
镍铬合金1.1 × 10⁻⁶合金

对比数据：镍铬合金电阻率约为铜的 65,000 倍，这正是电热丝选用镍铬合金而非铜丝的原因。
影响电阻的四个因素
  1. 材料 — 不同材料电阻率不同，是根本原因
  2. 长度 — 导体越长，电阻越大（正比关系）
  3. 横截面积 — 截面越大，电阻越小（反比关系）
  4. 温度 — 金属电阻随温度升高而增大；半导体反之
03
变阻器
RHEOSTAT
滑动变阻器原理

通过改变接入电路中电阻丝的长度来改变电阻值。铭牌上的 "20Ω 2A" 表示：最大阻值 20Ω，允许通过的最大电流 2A。

四种接法图示

正确接法 — 一上一下
接法A：A-C（左下+左上） ┌──────────────┐ │ A ═══════ C │ ← 滑片P向右 → R↑ │ │════│ │ [电阻丝] │ B D │ └──────────────┘ 接入：AP段电阻丝（左段） 接法B：B-D（右下+右上） ┌──────────────┐ │ A ════ C │ │ │════│ │ [电阻丝] │ B ═══════ D │ ← 滑片P向右 → R↓ └──────────────┘ 接入：BP段电阻丝（右段）
错误接法 — 警示
接法C：A-B（两下） ┌──────────────┐ │ A ═══════ B │ ← 全部电阻丝接入 │ │════│ │ [定值电阻] │ C D │ 滑片P失效 └──────────────┘ 结果：R = Rmax，无法改变 接法D：C-D（两上） ┌──────────────┐ │ A B │ │ │════│ │ │ C ═══════ D │ ← 金属杆接入 └──────────────┘ 结果：R ≈ 0，短路风险
两上接法 (C-D) 将金属杆接入电路，电阻几乎为零，可能造成短路。两下接法 (A-B) 将全部电阻丝接入，滑片失去调节作用。
使用规范
  1. 闭合开关前，滑片应置于阻值最大端（保护电路）
  2. 必须采用一上一下接线方式
  3. 电流不得超过铭牌标注的最大允许值
  4. 接入电路的接线柱决定有效电阻段，与滑片移动方向共同决定阻值变化趋势
04
欧姆定律
OHM'S LAW
I = U / R
电流 = 电压 / 电阻
定律内容

导体中的电流，跟导体两端的电压成正比，跟导体的电阻成反比。

适用范围：纯电阻电路（电能全部转化为内能的电路），金属导体和电解液导电均适用。不适用于气体导电、半导体元件等。

公式变形

U = I · R
已知电流、电阻求电压
R = U / I
已知电压、电流求电阻
注意：R = U/I 是电阻的计算式（量度式），不是决定式。电阻由导体自身因素决定，不能理解为 "电阻与电压成正比、与电流成反比"。

05
串并联电阻
CIRCUIT NETWORK
串联电路
R总 = R₁ + R₂ + ... + Rₙ

串联相当于增加导体长度，总电阻大于任一电阻。

┌───R₁───R₂───R₃───┐ │ │ └───────电源───────┘ I₁ = I₂ = I₃ = I总 U总 = U₁ + U₂ + U₃ U₁/U₂ = R₁/R₂
并联电路
1/R总 = 1/R₁ + 1/R₂ + ... + 1/Rₙ

并联相当于增加横截面积，总电阻小于任一电阻。

┌───R₁───┐ ──┬──┤ ├──┬── │ └───R₂───┘ │ │ └───R₃───┘ │ └─────电源─────┘ U₁ = U₂ = U₃ = U总 I总 = I₁ + I₂ + I₃ I₁/I₂ = R₂/R₁
两个电阻并联的快捷公式
R总 = (R₁ · R₂) / (R₁ + R₂)

仅适用于两个电阻并联，n 个电阻并联需用倒数和公式。

对比数据：串并联效果

R₁=4Ω, R₂=6Ω 时的等效电阻 DAT-02
连接方式公式计算过程R总
串联R₁ + R₂4 + 6 = 1010Ω
并联R₁R₂/(R₁+R₂)4×6 / (4+6) = 24/102.4Ω
串联：R总 > max(R₁, R₂)；并联：R总 < min(R₁, R₂)。n 个相同电阻 R 并联：R总 = R/n。
06
伏安法测电阻
VOLT-AMP METHOD
实验原理

用电压表测出待测电阻两端的电压 U，用电流表测出通过它的电流 I，由 R = U/I 计算电阻值。

两种接法对比

内接法 — 电流表内接
电路图： ┌────A────┐ │ │ │ 电源 Rₓ V │ │ │ └─────────┘ 电压表测量：U = Uᵣₓ + Uₐ 测得电阻：R测 = (Uᵣₓ+Uₐ)/I = Rₓ + Rₐ 误差：测量值偏大 适用：Rₓ >> Rₐ （大电阻）
外接法 — 电流表外接
电路图： ┌────Rₓ───┐ │ │ │ 电源 ├─A─┤ │ V │ └─────────┘ 电流表测量：I = Iᵣₓ + Iᵥ 测得电阻：R测 = U/(Iᵣₓ+Iᵥ) < Rₓ 误差：测量值偏小 适用：Rₓ << Rᵥ （小电阻）
口诀："大内偏大，小外偏小" — 大电阻用内接法测量值偏大，小电阻用外接法测量值偏小。

误差分析

系统误差来源
  1. 电表内阻 — 电流表分压 / 电压表分流，导致测量值偏离真实值
  2. 电表精度等级 — 指针式电表存在视差和摩擦误差
  3. 温度影响 — 通电发热导致电阻丝温度升高，电阻变化
  4. 接触电阻 — 接线柱、导线接头存在额外电阻
减小误差的方法
  • 多次测量取平均值，减小偶然误差
  • 根据 Rₓ 大小正确选择内接/外接法
  • 电流不宜过大，测量时间尽量短（减小温升）
  • 读数时视线垂直刻度盘，消除视差
  • 检查接线柱是否拧紧，减小接触电阻

实验步骤清单

  1. 按电路图连接实物，开关断开，滑片置于阻值最大端
  2. 检查电表量程、正负接线柱
  3. 闭合开关，调节滑动变阻器，记录第一组 U、I
  4. 改变滑片位置，获取 3~5 组不同数据
  5. 断开开关，整理器材
  6. 计算每组 R = U/I，求平均值
  7. 分析误差来源，撰写实验报告

数据处理方法

算术平均法
R̄ = (R₁ + R₂ + R₃) / 3

简单直观，适用于线性关系良好的数据。

图像法

绘制 U-I 图像，拟合直线求斜率。

R = ΔU / ΔI = 斜率

优点：可剔除异常数据，结果更可靠。
