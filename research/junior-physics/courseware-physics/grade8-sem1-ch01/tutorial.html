长度的测量

第 1 / 4 单元
核心工具 — 刻度尺

使用刻度尺前，先认清三个关键信息：

• 零刻度线：测量的起点，注意是否磨损

• 量程：刻度尺能测量的最大长度

• 分度值：最小一格代表的长度（如 1mm）

读数方法

• 视线与刻度尺刻度线垂直

• 读数要估读到分度值的下一位

• 记录结果 = 准确值 + 估读值 + 单位

如分度值 1mm → 读到 0.1mm（如 3.25 cm）

📌 例题 1：刻度尺分度值 1mm，物体左端 0.00 cm，右端 3.25 cm，长度是多少？
长度 = 末端读数 − 起始读数 = 3.25 cm − 0.00 cm = 3.25 cm
（注意：3.25 cm 的最后一位"5"是估读值）
📌 例题 2：物体左端对齐 1.00 cm，右端对齐 6.00 cm，长度是多少？
长度 = 6.00 cm − 1.00 cm = 5.00 cm（或写作 5.0 cm）
（注意：必须保留到分度值的下一位！不能写 5 cm）
📌 例题 3：物体左端 0.00 cm，右端 2.80 cm，长度是多少？
长度 = 2.80 cm − 0.00 cm = 2.80 cm
（末尾的"0"是估读位，不能省略！写 2.8 cm 就错了）

质量的测量

第 2 / 4 单元
核心工具 — 托盘天平

使用天平的核心口诀：左物右码

• 左盘放被测物体，右盘放砝码

• 砝码用镊子夹取，不能用手直接拿

• 添加砝码从大到小，最后调节游码

砝码规格 & 游码读数

常见砝码组合：100g、50g、20g、10g、5g

游码读数：看游码左侧对齐的刻度线

总质量 = 砝码质量之和 + 游码读数

📌 例题 1：右盘有 100g + 50g 砝码，游码读数 0.0g，物体质量是多少？
砝码总和 = 100 + 50 = 150 g
游码读数 = 0.0 g
→ 物体质量 = 150 + 0.0 = 150.0 g
（注意：游码在零刻度时，读数为 0）
📌 例题 2：右盘有 20g + 10g + 5g 砝码，游码读数 2.4g，总质量是多少？
砝码总和 = 20 + 10 + 5 = 35 g
游码读数 = 2.4 g
→ 总质量 = 35 + 2.4 = 37.4 g
（游码读数要精确到 0.1g）
📌 例题 3：右盘有 100g + 20g + 10g 砝码，游码读数 3.6g，总质量是多少？
砝码总和 = 100 + 20 + 10 = 130 g
游码读数 = 3.6 g
→ 总质量 = 130 + 3.6 = 133.6 g
（逐个相加砝码，别漏了游码！）

体积的测量

第 3 / 4 单元
量筒读数

量筒是测量液体体积的工具，读数时要注意：

• 视线与凹液面最低处保持水平

• 俯视读数会偏大，仰视读数会偏小

• 读数精确到分度值的下一位

排水法测不规则固体体积

对于形状不规则的固体（如石头），可以用排水法：

• 先在量筒中倒入适量水，记录体积 V₁

• 将固体完全浸入水中，记录体积 V₂

固体体积 V = V₂ − V₁

放入物体前

V₂
放入物体后
固体体积 = V₂ − V₁（视线对准凹液面最低处）
📌 例题 1：量筒中原有水 30.0 mL，放入石块后水面升到 42.0 mL，石块体积是多少？
V₁ = 30.0 mL，V₂ = 42.0 mL
石块体积 = V₂ − V₁ = 42.0 − 30.0 = 12.0 cm³
（1 mL = 1 cm³，注意单位换算）
📌 例题 2：量筒中原有水 25.0 mL，放入金属块后水面升到 38.5 mL，金属块体积是多少？
V₁ = 25.0 mL，V₂ = 38.5 mL
金属块体积 = 38.5 − 25.0 = 13.5 cm³
（减法时注意小数位对齐）
📌 例题 3：量筒中原有水 50.0 mL，放入小铁球后水面升到 56.0 mL，铁球体积是多少？
V₁ = 50.0 mL，V₂ = 56.0 mL
铁球体积 = 56.0 − 50.0 = 6.0 cm³
（小体积物体也能准确测量）

密度的计算

第 4 / 4 单元
核心公式

密度是物质的质量与体积之比，是物质的特性。

ρ = m / V

单位：g/cm³（常用）或 kg/m³

密度可以用来鉴别材料：不同物质密度一般不同。

📌 例题 1：一块金属质量 54.0 g，体积 20.0 cm³，求密度并判断是什么金属？
ρ = m / V = 54.0 / 20.0 = 2.7 g/cm³
查表：铝的密度 = 2.7 g/cm³
→ 这块金属是铝 ✅
（密度是物质的"身份证"）
📌 例题 2：一个铁块质量 158 g，体积 20 cm³，求密度。是纯铁吗？
ρ = 158 / 20 = 7.9 g/cm³
查表：铁的密度 = 7.9 g/cm³
→ 是纯铁 ✅
（计算结果与参考值完全吻合）
📌 例题 3：一杯水质量 100 g，体积 100 cm³，求密度。
ρ = 100 / 100 = 1.0 g/cm³
查表：水的密度 = 1.0 g/cm³
→ 确实是水 ✅
（水的密度是基准值，记住它！）
