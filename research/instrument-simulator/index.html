🎯 项目概述

乐器模拟器是一个探索性的研究项目，旨在通过现代 Web 技术重现中国传统乐器的独特音色与演奏体验。项目首期聚焦于编钟与古琴两种具有代表性的中国古代乐器。

通过 Web Audio API 的音频合成技术，我们将尝试：

  • 还原编钟的"一钟双音"特性与金属共鸣音色
  • 模拟古琴的散音、泛音、按音三种基本音色
  • 实现可交互的数字演奏界面
  • 提供基础的乐曲演奏功能

🎹 研究对象

🔔编钟 (Bianzhong)

历史背景：编钟是中国古代大型打击乐器，兴起于西周，盛于春秋战国至秦汉。1978年出土的曾侯乙编钟是迄今世界上规模最大、音乐性能最好的青铜编钟，被誉为"古代世界第八大奇迹"。

构造特点：由青铜铸造的钟按音阶排列，悬挂在钟架上。每件钟可发出两个不同音高（正鼓音、侧鼓音），形成独特的"一钟双音"现象。

音色特征：

  • 清脆悦耳，金属质感浓厚
  • 大钟深沉雄浑，小钟明亮清脆
  • 延音悠长，共鸣丰富
  • 音域宽广（可达五个八度）

演奏技巧：使用木质丁字形击槌敲击钟体不同部位，技法包括单击、双击、滚奏、弹击、摇击、顿击、闷击、扫击等。

代表曲目：《东方红》《楚殇》《春江花月夜》《欢乐颂》《交响曲1997：天·地·人》

打击乐器 青铜铸造 一钟双音 宫廷礼乐

🎼古琴 (Guqin)

历史背景：古琴是中国最古老的弹拨乐器之一，有三千年以上历史，位列"琴棋书画"四艺之首。2003年被联合国教科文组织列为"人类口头和非物质遗产代表作"。

构造特点：由桐木或杉木制成的狭长形音箱，七根丝弦或钢弦，十三个徽位标记泛音位置。琴身长约120-125厘米，宽约20厘米。

三种基本音色：

  • 散音：直接拨动空弦，音色清亮纯净，类似钟声
  • 泛音：轻触徽位后拨弦，音色空灵飘渺，如山谷回音
  • 按音：左手按弦取音，音色柔和温润，可塑性强

演奏技巧：右手基本指法有"抹、挑、勾、剔、打、摘、擘、托"八法，左手有吟、猱、绰、注等技法，讲究"按欲入木，弹欲断弦"。

十大名曲：《高山流水》《广陵散》《平沙落雁》《梅花三弄》《渔樵问答》《阳春白雪》《胡笳十八拍》《阳关三叠》《醉渔唱晚》《潇湘水云》

弹拨乐器 七弦琴 文人雅器 世界非遗

⚙️ 技术方案

音频合成技术

编钟音色合成：

  • 基础波形：使用正弦波叠加生成钟体基频
  • 谐波结构：添加低频谐波模拟青铜钟的厚重感
  • 包络控制：快速Attack（击钟瞬间）+ 长Decay（余音绵长）
  • 一钟双音：通过频率调制实现正鼓音与侧鼓音
  • 空间效果：添加混响模拟钟体共鸣与空间感

古琴音色合成：

  • 散音：锯齿波 + 低通滤波，模拟弦的基频振动
  • 泛音：正弦波高频成分，快速衰减
  • 按音：三角波 + 动态滤波器，模拟按弦的摩擦感
  • 吟猱效果：使用LFO调制音高，模拟左手揉弦
  • 走手音：滑音处理，模拟左手在弦上移动

技术栈

  • 核心 API：Web Audio API
  • 前端框架：原生 HTML5 + CSS3 + JavaScript
  • 音频处理：OscillatorNode, GainNode, BiquadFilterNode, ConvolverNode
  • 交互设备：支持鼠标、键盘、触摸、MIDI键盘
