## 二、第5章 · 比与比例

### 2.1 教材内容映射

| 小节 | 知识点 | 特工任务映射 |
|------|--------|-------------|
| 5.1 比、比例及其性质 | 比的概念、比的基本性质、比例、比例的基本性质、比例尺 | 特工配方解密、地图缩放术 |
| 5.2 百分数 | 百分数意义、百分数与小数/分数互化、百分数应用题 | 情报破译率、任务成功率计算 |

### 2.2 教案设计（tutorial.html）

**章节主题：** 🧪 资源调配部入职培训

**5个教学模块：**

1. **特工配方基础** — 比的概念
   - 动画：两种试剂按 2:3 混合，液面实时按比例上升
   - 交互：拖拽滑块改变配比，观察总量变化

2. **配方放大与缩小** — 比的基本性质
   - 动画：配方从 2:3 等比放大到 4:6、6:9，液柱同步缩放
   - 核心结论：前项后项同乘同除，比值不变

3. **跨基地调配** — 比例与比例尺
   - 交互地图：比例尺 1:10000 的基地平面图
   - 任务：测量图上距离，计算实际距离
   - Canvas 动画：缩放镜头从平面图拉大到实景

4. **任务完成率** — 百分数
   - 进度条动画：任务完成率 75%、成功率 92%
   - 交互：百分数 ↔ 小数 ↔ 分数 三者互化拖拽配对

5. **特工集训总结** — 公式卡
   - `a:b = a÷b = a/b`
   - `a:b = c:d → ad = bc`
   - `图上距离 / 实际距离 = 比例尺`

**随堂测验（3题）：**
1. 化简比 24:36
2. 解比例 3:5 = x:20
3. 地图上 5cm 对应实际 2km，求比例尺

## 三、第6章 · 圆与扇形

### 3.1 教材内容映射

| 小节 | 知识点 | 特工任务映射 |
|------|--------|-------------|
| 6.1 圆的周长与弧长 | 圆周率、周长公式 C=πd=2πr、弧长公式 l=nπr/180 | 圆形基地巡逻路线规划 |
| 6.2 圆与扇形的面积 | 圆面积 S=πr²、扇形面积 S=nπr²/360 | 扇形防御区域覆盖计算 |

### 3.2 教案设计（tutorial.html）

**章节主题：** 🎯 几何特勤组战术训练

**5个教学模块：**

1. **圆周追踪** — 圆周率与周长
   - Canvas 动画：圆滚动一周，直径标记，π≈3.14 的可视化
   - 交互：改变半径，实时计算周长并动画展示

2. **弧长测绘** — 圆心角与弧长
   - 动画：扇形角度从 0° 到 360°，弧长实时变化
   - 交互：拖拽圆心角滑块，显示弧长公式动态计算

3. **区域覆盖** — 圆面积
   - 动画：圆分割成扇形，拼接近似长方形（推导 πr²）
   - 交互：改变半径，面积实时变化

4. **扇形阵地** — 扇形面积
   - 动画：圆形基地划分多个扇形防御区
   - 交互：给定半径和圆心角，计算扇形面积

5. **战术总结** — 公式卡
   - `C = πd = 2πr`
   - `l = (n/360) × 2πr = nπr/180`
   - `S圆 = πr²`
   - `S扇 = (n/360) × πr²`

**随堂测验（3题）：**
1. 半径 5cm 的圆周长
2. 圆心角 60°、半径 6cm 的扇形弧长
3. 半径 4cm 的圆面积

## 四、第7章 · 可能性与统计图表

### 4.1 教材内容映射

| 小节 | 知识点 | 特工任务映射 |
|------|--------|-------------|
| 7.1 随机现象及其结果的可能性 | 确定事件/随机事件/不可能事件、可能性大小比较 | 任务风险评估 |
| 7.2 数据的收集、整理与表达 | 数据收集方法、条形图/折线图/扇形图 | 情报数据可视化 |
| 7.3 百分数的统计意义 | 频率估计概率、百分数在统计中的应用 | 任务成功率预测 |

### 4.2 教案设计（tutorial.html）

**章节主题：** 📡 情报分析科特训

**5个教学模块：**

1. **事件分类室** — 确定/随机/不可能事件
   - 交互：10个任务场景卡片，拖拽到「确定/随机/不可能」三个区域
   - 动画：分类正确时卡片发光，错误时红色抖动

2. **可能性评估** — 可能性大小
   - 交互转盘：4色转盘，点击旋转，记录落点频率
   - 实时统计：随着试验次数增加，频率趋近理论概率

3. **数据搜集站** — 数据收集与整理
   - 模拟：特工汇报各地点敌方数量，学生填入表格
   - 交互：拖拽数据到正确区间，自动生成分组表

4. **图表解码室** — 统计图表
   - 三种图表切换：条形图（数量对比）、折线图（趋势）、扇形图（占比）
   - 交互：同一组数据，点击切换图表类型，观察特点
   - Canvas 动画：数据实时绘制图表

5. **情报总结** — 公式卡
   - `频率 = 发生次数 / 总次数`
   - `概率 ≈ 大量重复试验后的频率`
   - 扇形图圆心角 = 百分比 × 360°

**随堂测验（3题）：**
1. "明天太阳从西边升起"属于什么事件？
2. 抛硬币 100 次，正面 48 次，正面频率是多少？
3. 某班 40 人，喜欢足球 25%，画扇形图圆心角多少度？

## 五、第8章 · 圆柱与圆锥

### 5.1 教材内容映射

| 小节 | 知识点 | 特工任务映射 |
|------|--------|-------------|
| 8.1 圆柱及其侧面展开图 | 圆柱特征、侧面展开为长方形、表面积、体积 V=πr²h | 圆柱形仓库设计 |
| 8.2 圆锥及其侧面展开图 | 圆锥特征、侧面展开为扇形、表面积、体积 V=πr²h/3 | 圆锥形信号塔设计 |

### 5.2 教案设计（tutorial.html）

**章节主题：** 🏗️ 立体工程队训练营

**5个教学模块：**

1. **圆柱解体** — 圆柱侧面展开
   - 3D Canvas 动画：圆柱旋转，侧面展开为长方形
   - 标注：长方形长 = 底面周长 = 2πr，宽 = 高 h
   - 交互：改变半径/高，观察展开图变化

2. **圆柱表面积** — 全面积计算
   - 动画：底面圆 × 2 + 侧面长方形，三块拼成总表面积
   - 公式推导：`S表 = 2πr² + 2πrh`
   - 交互：输入 r 和 h，实时计算并可视化各部分面积

3. **圆柱容积** — 体积公式
   - 动画：圆柱切割成无数薄圆片，堆叠演示体积
   - 与长方体体积对比：`V = 底面积 × 高`
   - 交互：水位上升动画，放入圆柱后体积置换

4. **圆锥解体** — 圆锥侧面展开
   - 3D Canvas 动画：圆锥旋转，侧面展开为扇形
   - 标注：扇形半径 = 母线 l，弧长 = 底面周长 2πr
   - 关键对比：同底同高的圆锥与圆柱体积关系

5. **工程总结** — 公式卡
   - `S圆柱侧 = 2πrh`
   - `S圆柱表 = 2πr² + 2πrh`
   - `V圆柱 = πr²h`
   - `V圆锥 = (1/3)πr²h`

**随堂测验（3题）：**
1. 圆柱半径 3cm，高 5cm，求侧面积
2. 圆锥底面半径 4cm，高 6cm，求体积
3. 圆柱和圆锥等底等高，圆柱体积 36cm³，圆锥体积多少？
