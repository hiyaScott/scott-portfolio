2核心概念

🎯 正数与负数

大于 0 的数叫做正数，如 +5、+3.2、+1/2。

在正数前面加上"-"号的数叫做负数，如 -3、-0.5、-2/3。

0 既不是正数，也不是负数。

💡 关键理解：正负数可以表示具有相反意义的量。
例如：收入 +100 元 vs 支出 -50 元；零上 +5°C vs 零下 -3°C

📏 数轴

规定了原点、正方向和单位长度的直线叫做数轴。

数轴上，右边的数总比左边的数大。
正数 > 0 > 负数

🪞 相反数

只有符号不同的两个数互为相反数。

a 的相反数是 -a
0 的相反数是 0

📐 绝对值

数轴上表示数 a 的点到原点的距离叫做 a 的绝对值，记作 |a|。

|正数| = 它本身
|负数| = 它的相反数（即正数）
|0| = 0

3运算法则

➕➖ 加法与减法

加法法则

• 同号相加：取相同符号，绝对值相加
  (-3) + (-5) = -8
• 异号相加：取绝对值大的数的符号，绝对值相减
  7 + (-4) = 3
• 互为相反数相加 = 0
  5 + (-5) = 0

减法转化：减去一个数 = 加上这个数的相反数
a - b = a + (-b)

✕➗ 乘法与除法

乘法法则

• 同号得正，异号得负
  (-3) × (-4) = 12
  (-5) × 6 = -30
• 任何数与 0 相乘 = 0

除法转化：除以一个数 = 乘以这个数的倒数
a ÷ b = a × (1/b)  (b ≠ 0)

🔢 乘方

n 个相同因数 a 相乘，记作 aⁿ，读作"a 的 n 次方"或"a 的 n 次幂"。

乘方规律

• 负数的偶次幂是正数：(-2)⁴ = 16
• 负数的奇次幂是负数：(-2)³ = -8
• 正数的任何次幂都是正数
• 0 的任何正整数次幂都是 0

4混合运算顺序

① 先算乘方
② 再算乘除（从左到右）
③ 最后算加减（从左到右）
④ 有括号的先算括号里面的

例题

(-2)³ + 5 × (-3) - (-4)² ÷ 2
= -8 + (-15) - 16 ÷ 2
= -8 - 15 - 8
= -31
