正负号的秘密

第 1 / 5 章
核心定义

大于 0 的数叫正数，如 +5、+3.2。

在正数前面加"-"号的数叫负数，如 -3、-0.5。

0 既不是正数，也不是负数。

正数 > 0 > 负数
实际意义

正负数可以表示相反意义的量：

• 零上 5°C → +5 　零下 3°C → -3

• 存入 500 元 → +500 　支出 200 元 → -200

• 向东 10 米 → +10 　向西 8 米 → -8

零上和零下是相反意义的量，零上用正数表示，零下用负数表示。
→ 零下 3°C 记作 -3 ✅
📌 例题 2：存入 1000 元记作 +1000，支出 500 元记作？
存入和支出是相反意义的量，存入用正数，支出用负数。
→ 支出 500 元记作 -500 ✅

0 既不是正数也不是负数。

数轴定位

第 2 / 5 章
核心定义

规定了原点、正方向、单位长度的直线叫做数轴。

数轴上，右边的数总比左边的数大
-4 -3 -2 -1 0 +1 +2
特工当前位置：-3 　目标位置：原点
关键规律

• 正数在原点的右边

• 负数在原点的左边

• 0 是正负数的分界点

• 离原点越远，绝对值越大

📌 例题 1：在数轴上标出 -2、0、+3 的位置，并比较大小。
数轴从左到右：-2 → 0 → +3
→ -2 < 0 < +3 ✅
（右边的数总比左边大）
📌 例题 2：数轴上，-5 和 -2 哪个更大？
在数轴上，-2 在 -5 的右边。
→ -2 > -5 ✅
（负数比较：绝对值小的反而大）

镜像密码

第 3 / 5 章
核心定义

只有符号不同的两个数互为相反数。

a 的相反数是 -a

0 的相反数是 0。

🪞 镜像对称演示
+3
原点右侧 3 格
⇄
-3
原点左侧 3 格
+3 和 -3 到原点的距离相等，符号相反 → 互为相反数
注意区分

• 相反数：只有符号不同（如 5 和 -5）

• 倒数：乘积为 1（如 2 和 1/2）→ 这是另一个概念！

📌 例题 1：7 的相反数是？
相反数只改变符号。
→ 7 的相反数是 -7 ✅
📌 例题 2：-(-5) = ?
"-"号表示取相反数，-5 的相反数是 5。
→ -(-5) = 5 ✅
（负负得正）
📌 例题 3：如果 a = -3，那么 -a = ?
-a 表示 a 的相反数，-3 的相反数是 3。
→ -a = 3 ✅
（注意：-a 不一定是负数！）

测距系统

第 4 / 5 章
核心定义

数轴上表示数 a 的点到原点的距离叫做 a 的绝对值，记作 |a|。

|正数| = 它本身 　 |负数| = 它的相反数 　 |0| = 0
📏 距离测量
-5
← 5 格 →
┅┅┅┅┅
0
原点
┅┅┅
+3
← 3 格 →
|-5| = 5（距离原点 5 格） 　 |+3| = 3（距离原点 3 格）
重要性质

• 绝对值总是非负的（≥ 0）

• 互为相反数的两个数绝对值相等：|5| = |-5| = 5

• 比较两个负数：绝对值小的反而大（如 -2 > -5）

📌 例题 1：|-7| = ? 　 |+4| = ? 　 |0| = ?
|-7| = 7（负数的绝对值是它的相反数）
|+4| = 4（正数的绝对值是它本身）
|0| = 0
→ 7、4、0 ✅
📌 例题 2：比较 -3 和 -8 的大小。
|-3| = 3，|-8| = 8
3 < 8，绝对值小的反而大。
→ -3 > -8 ✅
