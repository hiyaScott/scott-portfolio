分数的意义与基本性质

第 1 / 5 章
核心定义

把单位"1"平均分成若干份，表示这样的一份或几份的数，叫做分数。

34 表示把单位"1"平均分成 4 份，取其中的 3 份
分数与除法的关系

分数可以看作是两个整数相除的结果：

a ÷ b = ab　(b ≠ 0)

被除数 → 分子　　除数 → 分母

可视化：一个披萨被切成 4 等份
绿色 = 取走的份数　|　灰色 = 剩余的份数
分数的基本性质

分数的分子和分母同时乘或除以同一个不为0的数，分数的大小不变。

23 = 2×43×4 = 812
1218 = 12÷618÷6 = 23
📌 例题 1：将 23 的分子分母同乘 4
23 = 2×43×4 = 812
分子分母同乘 4，分数大小不变。
📌 例题 2：将 1218 约分到最简
先找最大公约数：GCD(12, 18) = 6
1218 = 12÷618÷6 = 23
分子分母同除以最大公约数 6，得到最简分数。

分数的加减法

同分母加减

分母不变，分子相加减

ac ± bc = a±bc
异分母加减

先通分（找最小公倍数），再按同分母计算

带分数加减

整数部分和分数部分分别加减

如果分数部分不够减，从整数部分借 1

分子乘分子，分母乘分母

分数与小数互化

第 5 / 5 章
分数化小数

用分子 ÷ 分母

34 = 3 ÷ 4 = 0.75
小数化分数

看小数位数，写成分母为 10、100、1000... 的分数，再约分

0.25 = 25100 = 14
0.625 = 6251000 = 58
循环小数

有限小数：小数位数有限（如 0.5、0.75）

无限循环小数：小数部分有数字无限重复（如 1/3 = 0.333... = 0.3̄）

注意：一个最简分数，如果分母的质因数只有 2 和 5，就能化成有限小数；否则是无限循环小数。

分数化小数：38 = ?
3 ÷ 8 = 0.375
→ 3/8 = 0.375（有限小数）
分母 8 = 2³，质因数只有 2 → 有限小数 ✅
小数化分数：0.625 = ?
0.625 有 3 位小数 → 分母是 1000
0.625 = 6251000
约分：625÷125=5，1000÷125=8
→ 0.625 = 5/8 ✅
循环小数：13 = ?
1 ÷ 3 = 0.333333...
→ 1/3 = 0.3̄（无限循环小数）
分母 3 的质因数不是 2 或 5 → 无限循环小数
📌 例题 1：38 = ?（小数）
3 ÷ 8 = 0.375
→ 3/8 = 0.375 ✅
（有限小数，分母 8=2³ 只有质因数 2）
📌 例题 2：0.625 = ?（分数）
0.625 = 625/1000 = (625÷125)/(1000÷125) = 5/8
→ 0.625 = 5/8 ✅
（有3位小数→分母1000，再约分到最简）
📌 例题 3：13 = ?（小数）
1 ÷ 3 = 0.333... = 0.3̄
→ 1/3 = 0.3̄（无限循环小数）✅
（分母3含有质因数3，不是2或5，所以是无限循环小数）
