分数的意义与基本性质

第 1 / 5 章
核心概念

把单位"1"平均分成若干份，表示这样的一份或几份的数叫做分数。

a ÷ b = ab (b ≠ 0)

分数与除法的关系：被除数 ÷ 除数 = 被除数/除数

分数的基本性质

分数的分子和分母同时乘或除以相同的数（0除外），分数的大小不变。

ab = a×cb×c = a÷cb÷c (c ≠ 0)
📌 例题 1：将 2/3 的分子分母同时乘 4
2/3 = (2×4)/(3×4) = 8/12
📌 例题 2：将 12/18 约分到最简分数
12和18的最大公因数是6
12/18 = (12÷6)/(18÷6) = 2/3
📌 例题 3：填空 3/7 = ?/21
分母 7→21 乘了3，分子也要乘3
3/7 = 9/21

分数的大小比较

第 2 / 5 章
比较法则

同分母分数：分子大的分数大。

同分子分数：分母小的分数大。

异分母分数：先通分（找最小公倍数作公分母），再比较分子。

真分数 < 1，假分数 ≥ 1

📌 例题 1：比较 3/5 和 4/7
通分：3/5 = 21/35，4/7 = 20/35
21/35 > 20/35 → 3/5 > 4/7
📌 例题 2：比较 5/6 和 7/8
通分：5/6 = 20/24，7/8 = 21/24
20/24 < 21/24 → 5/6 < 7/8
📌 例题 3：比较 3/4 和 5/4
同分母，比分子：3 < 5
→ 3/4 < 5/4（真分数 < 假分数）

分数的加减法

第 3 / 5 章
运算法则

同分母加减：分母不变，分子相加减。

异分母加减：先通分，再加减。

带分数加减：整数部分和分数部分分别相加减。

📌 例题 1：1/4 + 2/4
同分母，分母不变，分子相加
1/4 + 2/4 = 3/4
📌 例题 2：3/5 + 1/2
通分：3/5 = 6/10，1/2 = 5/10
6/10 + 5/10 = 11/10 = 1又1/10
📌 例题 3：2又1/3 + 1又1/6
整数：2+1=3
分数：1/3+1/6=2/6+1/6=3/6=1/2
→ 3又1/2

分数的乘除法

第 4 / 5 章
运算法则

分数乘法：分子乘分子，分母乘分母。能约分的先约分。

分数除法：除以一个数 = 乘它的倒数。

带分数乘除：先化为假分数，再计算。

📌 例题 1：2/3 × 3/4
先约分更简便：2/3×3/4=2/4=1/2
（分子2×3=6，分母3×4=12，6/12=1/2）
📌 例题 2：5/6 ÷ 2/3
除以2/3 = 乘3/2
5/6 × 3/2 = 15/12 = 5/4
📌 例题 3：1又1/2 × 2/5
带分数化假分数：1又1/2 = 3/2
3/2 × 2/5 = 6/10 = 3/5

分数与小数互化

第 5 / 5 章
转化方法

分数化小数：分子 ÷ 分母。

小数化分数：看小数位数，写成分母是10/100/1000...的分数，再约分。

有限小数：分母只含质因数2和5的分数。

循环小数：分母含其他质因数的分数。

📌 例题 1：3/8 = ?（小数）
3 ÷ 8 = 0.375 → 0.375
（分母8=2³，只含2，是有限小数）
📌 例题 2：0.625 = ?（分数）
三位小数 = 625/1000 = 5/8
（约分：625÷125=5, 1000÷125=8）
📌 例题 3：1/3 = ?（小数）
1 ÷ 3 = 0.333... → 0.3̇
（循环小数，3循环）
