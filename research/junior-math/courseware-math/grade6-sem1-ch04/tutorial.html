线段

第 1 / 5 章
核心概念

线段：有两个端点，长度可以测量。如线段 AB。

比较方法：度量法（用尺量）或叠合法（重合比较）。

中点：把线段分成相等两部分的点。

若 M 是 AB 中点，则 AM = MB = AB/2
📏 线段可视化
A——M——B（M是中点）
📌 例题 1：AB = 10cm，M 是 AB 中点，求 AM。
M 是中点 → AM = AB/2
AM = 10 ÷ 2 = 5cm
→ AM = 5cm ✅
📌 例题 2：C 在 AB 上，AC = 3cm，CB = 5cm，求 AB。
AB = AC + CB
AB = 3 + 5 = 8cm
→ AB = 8cm ✅

角的初步

第 2 / 5 章
角的定义

角：由公共端点的两条射线组成的图形。

顶点：两条射线的公共端点。

边：组成角的两条射线。

记作 ∠AOB 或 ∠O（顶点字母）
角的度量

单位：度（°），用 protractor（量角器）测量。

• 锐角：0° < 角 < 90°

• 直角：= 90°

• 钝角：90° < 角 < 180°

• 平角：= 180°

📐 角的分类
锐角
直角
钝角
📌 例题 1：一个角是 45°，它是什么角？
45° < 90°
→ 锐角 ✅
📌 例题 2：∠A = 120°，它是什么角？
90° < 120° < 180°
→ 钝角 ✅

角的计算

第 3 / 5 章
角的和差

两个角可以相加或相减。

∠AOC = ∠AOB + ∠BOC
∠AOB = ∠AOC - ∠BOC
互余与互补

互余：两角之和为 90°。

互补：两角之和为 180°。

若 ∠A + ∠B = 90°，则 ∠A 与 ∠B 互余
若 ∠A + ∠B = 180°，则 ∠A 与 ∠B 互补
📐 角的和演示
∠AOB = 30° + 45° = 75°
📌 例题 1：∠A = 35°，∠B = 55°，它们互余吗？
35° + 55° = 90°
→ 互余 ✅
📌 例题 2：∠A = 120°，求它的补角。
补角 = 180° - 120° = 60°
→ 60° ✅
📌 例题 3：∠AOC = 90°，∠AOB = 30°，求 ∠BOC。
∠BOC = ∠AOC - ∠AOB
= 90° - 30° = 60°
→ 60° ✅
