📉 边缘化深度分析 警示专区

🔴 泰登 - 最弱传说之谜

官方定位: 防御者/警戒T

致命缺陷:

1. 没有援护 - 作为坦克无法保护队友，只能保护自己

2. 警戒机制缺陷 - 个性警戒范围太小(正面1×3)，且只能警戒一次

3. 腿短笨重 - 移动3格，无法快速到位

4. 魔防极低 - 全游传说防御者中魔防最低，怕法师

5. 伤害能力有限 - 输出疲软，只能打控制

玩家评价: "设计的很弱，有理由怀疑是污染卡池的角色"、"铃兰F4之一"

结论: 除非真爱或等SP/专武加强，否则不建议投入资源

🟠 史诗角色的边缘化困境

炎之魔女: 低星刮痧，燃烧地形被火魔女露维塔完全上位替代

风暴神斧: 粉碎者竞争激烈，被奥古斯特/嘉西亚/拉维耶完全压制

红隼: 萌新过渡可用，但突袭者SSR太多，无上场空间

摧城/锐眼: 守望者弓手内卷严重，艾达/法卡尔/莉莉薇儿压制

黯心: 漂泊者光环无用武之地，被其他领袖光环替代

🟡 强度膨胀下的失业者

麦莎 - 全装甲麦莎、蔻蔻娜上位，SR坦克无竞争力

🏷️ 核心机制标签详解

T0级机制（改变战局）

再动/再行动 让单位再次行动，战略价值极高。代表：伊南娜、科尔、歌洛莉亚、爱莎、伊南娜·铃兰之剑

大光环 阵营核心技能，提供攻防增益。代表：迪塔利奥、歌洛莉亚、马格努斯、诺诺薇儿

无视援护 直接攻击脆皮核心。代表：科尔

T1级机制（体系核心）

召唤体系 召唤额外单位作战。代表：伊南娜（卫兵）、阿坎贝（暗影刺客）、霍玛（鹦鹉）

感染流 通过感染DOT造成百分比伤害。核心：茉茉，配合莉莉薇儿

推拉/改变地形 改变敌人位置，地形杀。代表：茉茉、席梦娜

免死/不屈 关键时刻保命。代表：萨曼莎、翠斯坦、帕米娜、爱莎

T2级机制（战术辅助）

易伤/减防 提升团队输出。代表：索菲亚（易伤2）、诺诺薇儿（易伤3）、马格努斯（降防）

减CD/回能 提升技能循环。代表：蔻蔻娜（减CD减豆）、塔埃尔（回能）

传送 快速位移。代表：伊南娜

击退 位移控制。代表：歌洛莉亚、拉维耶、列奥尼德

援护/协防 保护队友。代表：马格努斯、内尔伽勒、莉拉

🎯 推荐阵容搭配

伊利亚国家队（万能阵容）

成员: 迪塔利奥 + 伊南娜 + 索菲亚 + 伦伽勒 + 马格努斯 + 艾达/贝拉

特点：双光环加持，再动体系，输出与生存兼备，适合所有场景

感染流阵容

成员: 茉茉 + 莉莉薇儿 + 诺诺薇儿 + 伊南娜 + 索菲亚 + 科尔

特点：感染DOT百分比伤害，适合打高血量BOSS，如逆境之塔

物理爆发队

成员: 奥古斯特 + 歌洛莉亚 + 嘉西亚 + 伊南娜 + 马格努斯 + 蔻蔻娜

特点：极限物理输出，适合速刷和打桩，不动如山阵营

召唤流阵容

成员: 阿坎贝 + 伊南娜 + 霍玛 + 迪塔利奥 + 索菲亚 + 艾达

特点：人海战术，召唤物抗伤输出，适合持久战

新手开荒队

成员: 拉维耶 + 麦莎 + 法卡尔 + 萨曼莎 + 泽维尔 + 炎之魔女

特点：易获取，推图稳定，培养成本低，SR+R阵容
