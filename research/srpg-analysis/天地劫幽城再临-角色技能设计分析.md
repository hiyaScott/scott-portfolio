## 二、职业系统架构分析

### 2.1 六大职业体系

```
┌─────────────────────────────────────────────────────────────────┐
│                    天地劫职业克制关系图                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│    光 ◄────── 暗                                                │
│    │              │                                             │
│    │    冰 ◄───► 火                                             │
│    │         │                                                  │
│    └───────► 雷 ◄───────────────────────────────────────────┐   │
│                                                              │   │
│    【职业克制独立于属性克制】                                 │   │
│    羽士 ──────► 御风 (25%增伤)                                │   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 职业详细分析

| 职业 | 移动力 | 攻击距离 | 核心机制 | 定位 | 数量占比 |
|------|--------|----------|----------|------|----------|
| **侠客** | 3格 | 1格 | 物理近战、连击/追击 | 主力输出 | ~25% |
| **铁卫** | 3格 | 1格 | 护卫(替友军承伤)、反击 | 坦克 | ~15% |
| **咒师** | 3格 | 2格 | 法术远程、AOE | 魔法输出 | ~20% |
| **祝由** | 3格 | 2格 | 治疗、驱散、增益 | 辅助 | ~15% |
| **御风** | 5格 | 1格 | 轻功/飞行、无视护卫 | 刺客/机动 | ~15% |
| **羽士** | 3格 | 2格 | 对御风克制、暴击 | 远程物理 | ~10% |

#### 职业设计亮点分析

**1. 御风-羽士克制链**
- 御风高机动(5格)+无视护卫，天然克制后排
- 羽士作为"反制位"设计，专门克制御风(25%增伤)
- 形成"剪刀石头布"式的博弈关系

**2. 铁卫的护卫机制**
- 2格范围替友军承伤，创造"阵型"概念
- 配合地形形成防御纵深
- 反击机制提供被动输出

**3. 羽士的职业困境**
- 只能攻击2格距离(近战无法反击)
- 单体技能倍率偏低
- 设计反思：职业特色不够突出，需要加强

## 三、五内化蕴系统分析

### 3.1 系统概述

```
                    【五内化蕴路线】
                    
                         烈(攻)
                          │
    魂(辅) ◄─────────────┼─────────────► 迅(速)
                          │
                         神(防)
                          │
                         魔(法)
```

### 3.2 设计解析

| 路线 | 属性倾向 | 典型技能类型 | 适用职业 |
|------|----------|--------------|----------|
| **烈** | 物理攻击 | 单体高伤、连击 | 侠客、御风 |
| **魂** | 辅助/特殊 | 召唤、驱散、增益 | 祝由、咒师 |
| **迅** | 速度/机动 | 位移、先手、闪避 | 御风、侠客 |
| **神** | 防御/生存 | 护卫、减伤、回血 | 铁卫、祝由 |
| **魔** | 法术攻击 | 法术伤害、法防 | 咒师、祝由 |

### 3.3 设计亮点

**1. 分支选择与角色塑造**
- 同一角色可走不同路线，形成差异化Build
- 例如冰璃：上路(群伤) vs 下路(先攻护卫)

**2. 资源投入策略**
- 五蕴石稀缺资源，迫使玩家做选择
- 主力角色可投资双线，工具人单线即可

**3. 技能组合自由度**
- 解锁的技能可跨路线搭配(3个技能槽位)
- PVE/PVP需要不同技能组合

## 四、技能机制深度分析

### 4.1 技能类型分布

基于收集的角色数据，技能类型分布如下：

| 技能类型 | 占比 | 典型代表 | 设计作用 |
|----------|------|----------|----------|
| 单体伤害 | 35% | 避实就虚、天霜雪舞 | 核心输出手段 |
| 群体伤害(AOE) | 20% | 冰华之阵、千煌幻日 | 清场/控场 |
| 位移/机动 | 15% | 冰华飞刺、疾风突袭 | 创造战术空间 |
| 治疗/驱散 | 15% | 神华天舞、气愈之术 | 续航保障 |
| 增益/减益 | 10% | 神睿、减防 | 战术辅助 |
| 特殊机制 | 5% | 召唤、再动、免死 | 角色特色 |

### 4.2 核心机制解析

#### 4.2.1 再动机制

| 角色 | 再动条件 | 再动类型 | 战略价值 |
|------|----------|----------|----------|
| **冰璃** | 主动攻击后 | 自身再动 | 最高优先级，输出翻倍 |
| **封铃笙** | 主动技能 | 队友再动 | 战术核心，改变战局 |
| **李盈凤** | 主动绝学后 | 普攻再动 | 爆发输出 |

**设计评价**：再动是游戏最强机制之一，拥有再动的角色均为T0级。

#### 4.2.2 无视护卫机制

| 角色 | 技能 | 应用场景 |
|------|------|----------|
| **燕明蓉** | 疾风快刀、灵狐穿刺 | 切后排核心 |
| **曹沁** | 标记+瞬移 | 刺客天花板 |

**设计评价**：御风职业的立身之本，与铁卫形成直接对抗。

#### 4.2.3 先攻机制

| 角色 | 先攻条件 | 效果 |
|------|----------|------|
| **殷无邪** | 常驻 | 被攻击时先出手 |
| **冰璃** | 下路技能 | 护卫范围内先攻 |
| **杨云佐** | 有增益时 | 被攻击先攻 |

**设计评价**：改变攻防节奏，让防守方掌握主动权。

#### 4.2.4 召唤机制

| 角色 | 召唤物 | 机制特色 |
|------|--------|----------|
| **郸阴** | 灵俑 | 替死、承伤、输出 |
| **尉迟良** | 机关蜘蛛 | 工具人、引怪、卡位 |

**设计评价**：召唤物提供额外策略维度，但平衡性较难把控。

### 4.3 技能倍率分析

| 技能类型 | 倍率范围 | 备注 |
|----------|----------|------|
| 普攻 | 1.0x | 基础输出 |
| 单体技能 | 1.3x - 1.8x | 主力输出技能 |
| AOE技能 | 0.4x - 0.7x | 范围伤害，总伤高 |
| 连击/追击 | 0.3x - 0.5x | 附加伤害 |

**设计规律**：
- 倍率与CD/气力消耗挂钩
- 高倍率技能通常有使用限制(气力、血量代价)
- AOE倍率低但可攻击多个目标

## 五、角色设计案例深度分析

### 5.1 T0级角色解析

#### 5.1.1 冰璃 —— 完美设计范本

| 维度 | 分析 |
|------|------|
| **定位** | 冰系侠客，主C/副T |
| **核心机制** | 再动 + 免死 + 先攻护卫 |
| **技能组合** | 避实就虚(减攻) / 幽剑冥引(单体) / 定魂(免死) |

**设计亮点**：
1. **再动机制**：主动攻击后可再行动，输出效率翻倍
2. **定魂被动**：抵御一次致命伤害，容错率极高
3. **先攻护卫**：下路技能让她能客串坦克
4. **五内选择丰富**：上下两路都有价值，投资双线不亏

**设计启示**：多面手角色，一个角色承担多个职能，提高阵容灵活性。

---

#### 5.1.2 曹沁 —— 刺客天花板

| 维度 | 分析 |
|------|------|
| **定位** | 火系御风，刺客 |
| **核心机制** | 闪避 + 瞬移 + 标记 |
| **技能组合** | 劫焰誓杀(标记) / 追心影刃(高伤) / 闪避被动 |

**设计亮点**：
1. **标记机制**：标记敌人后可瞬移至身边，无视地形
2. **双闪避**：技能和被动各提供一次闪避，生存能力极强
3. **高爆发**：标记后伤害加成，配合无视护卫切后排

**设计启示**：刺客职业的标杆，高风险高回报，操作感强。

---

#### 5.1.3 封铃笙 —— 战术核心

| 维度 | 分析 |
|------|------|
| **定位** | 光系祝由，奶妈 |
| **核心机制** | 队友再动 + 驱散 + 群疗 |
| **技能组合** | 瞬霎神仪(再动) / 地御之阵(护卫) / 神华天舞(群疗) |

**设计亮点**：
1. **唯一再动奶**：让队友再行动一次，战术价值无法替代
2. **驱散能力强**：光系特性，可驱散debuff
3. **兼职铁卫**：可学习护卫技能，增加阵容灵活性

**设计启示**：辅助角色的设计不应局限于治疗，改变行动顺序的机制极具价值。

## 六、数值与平衡分析

### 6.1 属性克制系统

```
    光 ──────► 暗
    │            │
    ▼            ▼
    冰 ◄──────► 火
         │
         ▼
         雷

克制效果：+25%伤害
被克制：-25%伤害
```

**设计评价**：
- 五属性循环克制，光暗互克
- 克制效果适中(25%)，不至于碾压
- 鼓励玩家培养多属性队伍

### 6.2 职业-属性矩阵

| 职业 | 冰 | 火 | 雷 | 光 | 暗 |
|------|----|----|----|----|----|
| 侠客 | 冰璃、殷无邪 | 殷剑平 | 皇甫申 | 李逍遥 | - |
| 铁卫 | 阴歙 | - | 古伦德 | 宇韶容 | 解臾 |
| 咒师 | 封寒月、赵灵儿 | 夏侯仪 | 慕容璇玑、紫枫 | 诸葛艾 | 黎幽、郸阴 |
| 祝由 | 于小雪、傅雅鱼 | - | 葛云衣 | 封铃笙 | - |
| 御风 | - | 曹沁 | 燕明蓉、林月如 | - | 剑邪 |
| 羽士 | - | 韩千秀 | - | 李盈凤 | - |

**观察结论**：
- 侠客、咒师分布最均衡
- 暗系以铁卫/咒师/御风为主
- 光系稀少但质量高

### 6.3 技能CD与气力系统

| 技能类型 | CD范围 | 气力需求 | 设计意图 |
|----------|--------|----------|----------|
| 常规技能 | 2-3回合 | 0 | 主力输出循环 |
| 强力技能 | 3-5回合 | 0 | 爆发/控场 |
| 气力技 | 2-3回合 | 3-5点 | 限制高频使用 |

**气力获取方式**：
- 造成伤害：+1气力
- 受到伤害：+1气力
- 特殊天赋/技能额外获取

## 七、设计亮点与不足

### 7.1 设计亮点

1. **五内化蕴系统**
   - 同角色多Build可能性
   - 资源稀缺性创造选择压力
   - 技能跨路线搭配增加策略深度

2. **再动机制的运用**
   - 冰璃、封铃笙等核心角色围绕再动设计
   - 改变行动顺序创造战术空间
   - 高价值但不泛滥

3. **职业克制链**
   - 御风-羽士形成明确克制
   - 铁卫护卫创造阵型概念
   - 多种克制关系增加博弈

4. **战阵系统**
   - 特定角色组合触发
   - 增加阵容搭配深度
   - 鼓励收集特定角色

### 7.2 设计不足

1. **羽士职业弱势**
   - 无法近战反击
   - 单体倍率偏低
   - 职业特色不突出

2. **数值膨胀问题**
   - 后期角色机制越来越强
   - 早期角色逐渐退环境
   - 平衡性调整频率高

3. **PVE与PVP设计冲突**
   - 某些角色PVE强但PVP弱
   - 技能设计难以兼顾两种场景
   - 玩家需要准备两套配置

## 八、对《计策设计》的启示

### 8.1 可借鉴的设计元素

| 元素 | 天地劫实现 | 应用到计策设计 |
|------|-----------|----------------|
| **五内化蕴** | 分支路线选择 | 计策升级路线 |
| **再动机制** | 改变行动顺序 | 特殊计策效果 |
| **职业克制** | 25%伤害差 | 流派克制关系 |
| **无视护卫** | 切后排机制 | 穿透/破防计策 |

### 8.2 技能设计模式参考

```
【单体输出技能模板】
- 倍率：1.5x - 1.8x
- CD：2-3回合
- 附加效果：减攻/减防/眩晕

【AOE技能模板】
- 倍率：0.5x - 0.7x
- 范围：十字/菱形/直线
- 附加效果：减速/燃烧/冰冻

【辅助技能模板】
- 增益：攻击+20%、移动力+1
- 持续时间：2-3回合
- CD：3-4回合
```

### 8.3 流派设计参考

基于天地劫的分析，计策系统可考虑以下流派：

| 流派 | 核心机制 | 代表技能类型 |
|------|----------|--------------|
| **快攻流** | 高机动、先攻、连击 | 位移、追击 |
| **控制流** | 眩晕、减速、沉默 | 减益、状态 |
| **爆发流** | 高倍率、暴击、再动 | 单体高伤 |
| **持久战** | 吸血、回复、反击 | 续航、被动 |
| **战术流** | 召唤、再动、换位 | 特殊机制 |

## 十、总结

### 核心设计哲学

1. **策略优先于数值**：机制(再动、无视护卫)比单纯高数值更有价值
2. **选择产生差异**：五内化蕴让玩家自定义角色定位
3. **克制创造博弈**：属性+职业双重克制增加阵容深度
4. **特色角色记忆点**：每个T0角色都有独特的标志性机制
