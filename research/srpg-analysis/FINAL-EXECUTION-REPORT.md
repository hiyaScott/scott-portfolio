## 四、核心教训（已加入长期记忆）

### 错误根因
```
┌─────────────────────────────────────────────────────────────┐
│ 1. 单一来源依赖 - 使用攻略网站而非官方Wiki                    │
│ 2. 缺乏版本控制 - 未标注数据对应的游戏版本                    │
│ 3. 未验证录入   - 数据录入后未抽样核查                        │
```
