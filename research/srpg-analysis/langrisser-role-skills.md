## 二、转职系统深度分析

### 2.1 转职树架构

```
【转职树示例 - 艾尔文】

                    英雄 (终极)
                   /
    战士 ──► 领主 ──► 大领主 ──► 战士统帅
   (初始)                              \
                                        大元帅
                                         (坦克路线)
                    
    战士 ──► 骑士 ──► 骑士长 ──► 骑士统帅 ──► 皇家骑士
                                              (突击路线)
```

### 2.2 设计解析

| 设计要素 | 机制说明 | 战略价值 |
|----------|----------|----------|
| **路线分支** | 每个英雄3条转职路线 | 同英雄不同定位 |
| **精通继承** | 已精通职业的属性和技能保留 | 鼓励多线发展 |
| **魔导石** | 切换路线需要消耗 | 增加策略沉没成本 |
| **终极职业** | 每条路线终点有强力职业 | 给予培养目标 |

### 2.3 与天地劫五内化蕴的对比

| 对比项 | 梦幻模拟战转职 | 天地劫五内 |
|--------|---------------|-----------|
| **选择维度** | 职业切换(大方向) | 技能选择(小方向) |
| **属性变化** | 职业决定基础属性 | 路线决定属性加成 |
| **技能来源** | 职业自带技能 | 五内路线解锁 |
| **灵活性** | 切换有成本 | 解锁后自由搭配 |
| **深度** | 职业数量多，组合复杂 | 单角色Build丰富 |

**设计评价**：
- 梦幻的转职更适合**多角色收集**导向，鼓励练多个英雄
- 天地劫的五内更适合**单角色深挖**，一个英雄有多种玩法

## 三、佣兵系统深度分析

### 3.1 系统架构

```
【佣兵系统核心逻辑】

英雄(指挥官) + 佣兵(部队) = 战斗单位
      │                │
      │                ├─ 兵种类型(步兵/骑兵/枪兵等)
      │                ├─ 兵种等级(1-10级)
      │                └─ 兵种科技(练兵场强化)
      │
      └─ 职业决定可用兵种
      └─ 指挥范围影响佣兵属性
```

### 3.2 兵种体系

#### 基础兵种三角

```
        步兵
       /    \
      /      \
     /   克制  \
    /            \
 枪兵 ◄────────► 骑兵
  
克制效果：攻击+10 防御+10 (对克制目标)
```

#### 完整兵种列表

| 兵种类型 | 代表兵种 | 移动力 | 特点 | 克制关系 |
|----------|----------|--------|------|----------|
| **步兵** | 重装步兵 | 3-4格 | 高防御、稳推进 | 克枪兵 |
| **骑兵** | 皇家骑兵 | 5格 | 高机动、冲锋 | 克步兵 |
| **枪兵** | 重装枪兵 | 3-4格 | 反骑兵、阵地战 | 克骑兵 |
| **飞兵** | 天使、狮鹫 | 6格 | 无视地形、高机动 | 被弓兵克制 |
| **弓兵** | 精灵射手 | 3格 | 远程2格、高暴击 | 克飞兵 |
| **僧侣** | 神官、司祭 | 3格 | 治疗、克制魔物 | 克魔物/不死 |
| **魔物** | 石像鬼、恶魔 | 4-5格 | 高攻击、邪道 | 被僧侣克制 |
| **水兵** | 人鱼、海怪 | 4格(水) | 水域优势 | 陆地弱势 |
| **刺客** | 暗杀者 | 4格 | 暴击、闪避 | 脆皮 |

### 3.3 佣兵-英雄搭配策略

| 英雄类型 | 推荐佣兵 | 搭配理由 |
|----------|----------|----------|
| **突击型** | 骑兵/飞兵 | 高机动配合突进技能 |
| **防御型** | 枪兵/重步 | 阵地战、反击输出 |
| **魔法型** | 远程兵种 | 保持安全距离输出 |
| **辅助型** | 步兵/僧侣 | 生存优先，保护施法 |

### 3.4 指挥范围机制

```
【指挥范围示意图】

    . . . . .
    . ○ ○ ○ .
    . ○ 英 ○ .  ← 英雄周围3格为指挥范围
    . ○ ○ ○ .
    . . . . .
    
范围内佣兵获得英雄属性加成
加成比例随英雄职业/星级提升
```

**设计评价**：
- 佣兵系统让"单兵作战"变成"军团作战"，增加战场信息量
- 兵种克制让阵容搭配变得重要
- 指挥范围限制部队分散，保持阵型概念

## 八、设计亮点与不足

### 8.1 设计亮点

1. **佣兵系统的传承与创新**
   - 保留原作"军团作战"精髓
   - 增加兵种科技养成线
   - 英雄+佣兵双重养成

2. **转职树的选择深度**
   - 同一英雄多路线发展
   - 精通继承鼓励探索
   - 魔导石切换增加沉没成本

3. **原作情怀的还原**
   - 时空裂缝收录经典关卡
   - 多路线剧情复刻
   - 漆原智志原画风格

4. **超绝阵营体系**
   - 清晰的组队逻辑
   - 阵营认同感
   - 限制与自由度的平衡

### 8.2 设计不足

1. **后期数值膨胀**
   - 新英雄机制越来越复杂
   - 老英雄逐渐退环境
   - 节奏把控难度高

2. **PVP平衡挑战**
   - 巅峰竞技场强势阵容固化
   - 某些机制(再动)难以平衡
   - Ban/Pick系统复杂度高

3. **新手门槛**
   - 转职路线选择困难
   - 兵种搭配学习成本高
   - 缺乏引导易走弯路

### 9.3 综合建议

基于两款紫龙SRPG的分析，对《计策设计》的建议：

1. **计策升级系统**
   - 参考转职树：多分支路线选择
   - 参考五内化蕴：路线间自由搭配
   - 结合两者优势：既有方向感又有自由度

2. **单位系统**
   - 可考虑"主将+副将/部队"双层设计
   - 增加兵种/流派克制关系
   - 指挥范围限制部队分散

3. **组队逻辑**
   - 建立清晰的流派体系
   - 流派核心计策提供团队加成
   - 限制混合流派收益，鼓励专精

4. **技能设计**
   - 基础技能：高频使用，稳定输出
   - 特色技能：改变战局，有使用限制
   - 终极技能：高CD/高消耗，翻盘手段

## 十一、总结

### 核心设计哲学

1. **佣兵是梦幻的灵魂**
   - "军团作战"区别于其他SRPG
   - 兵种克制增加策略维度
   - 英雄+佣兵双重养成

2. **转职创造收集价值**
   - 同一英雄多路线=多次培养
   - 精通继承鼓励探索
   - 魔导石切换控制节奏

3. **超绝定义组队逻辑**
   - 清晰的阵营归属
   - 同阵营加成鼓励主题队伍
   - 限制与自由的平衡

4. **情怀是核心竞争力**
   - 原作关卡复刻
   - 经典BGM、原画
   - 多路线剧情还原

### 对计策设计的最终建议

| 方面 | 建议 |
|------|------|
| **计策升级** | 融合转职树的方向感+五内的自由度 |
| **单位系统** | 考虑主将+副将/召唤物的双层设计 |
| **流派体系** | 建立清晰的流派核心+组队加成 |
| **技能层级** | 基础技能+特色技能+终极技能三层 |
| **克制关系** | 明确的流派/属性克制链 |
