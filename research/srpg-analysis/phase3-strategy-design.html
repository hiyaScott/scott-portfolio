🎯 计策系统设计方案

第三阶段：基于183位英雄数据的计策系统核心设计建议

💡 核心设计理念

基于对4款主流SRPG（183位英雄/武将）的深度分析，结合战棋游戏设计与网络游戏系统架构的双重专业视角，提出以下计策系统核心设计观点：

🎲 核心洞察：计策的本质是"战中成长"

不同于被动技能，计策的本质是让玩家在战斗过程中持续成长、积累优势的能力。这是与传统SRPG最本质的差异——不是静态的数值比拼，而是动态的优势建构。

  • 数值累加型成长：每次行动叠加层数，后期爆发（如司马懿的"鹰视狼顾"）
  • 优势造成型成长：通过计策创造地形/时机优势，滚雪球扩大领先
  • 适应型成长：根据战场变化调整策略，越打越顺的掌控感

战中成长让玩家每一回合都有"我在变强"的正反馈，避免战棋常见的"开局定胜负"枯燥感。

设计层级框架：概念 → 结构 → 细节 → 体验

🎯 概念层（Why）

计策系统的存在意义

  • 解决战棋"开局定胜负"的枯燥感
  • 提供战中成长与变数
  • 让玩家掌握"改写规则"的权力
  • 与思考领域形成教学-应用闭环

🏗️ 结构层（What）

计策系统的骨架

  • 三大类计策：时机型/环境型/数值型
  • 成长维度：数值累加/优势造成/适应型
  • 限制体系：CD/能量/代价/条件
  • 解锁路径：思考领域 → 常规战斗

🔍 细节层（How）

具体数值与规则

  • 数值边界：单体≤1.8倍，AOE≤0.7倍
  • 持续时间：1-3回合最佳
  • 成长曲线：线性/指数/阶梯式
  • 触发条件：击杀/受击/回合结束等

✨ 体验层（Feel）

玩家的主观感受

  • 掌控感："局势在我手中"
  • 成长感："越打越强"
  • 惊喜感："关键时刻翻盘"
  • 成就感："策略被验证"

🎮 用户体验深度设计

基于"战中成长"的核心理念，从心流曲线、反馈设计、认知负荷三个维度深入设计用户体验：

1. 心流曲线：战中成长的节奏感

战斗节奏模型：
开局（资源积累） → 中盘（优势扩大） → 终盘（爆发收割）
↓ ↓ ↓
谨慎布局 主动出击 爽快感释放
阶段回合数玩家心理计策设计目标示例
开局期1-2回合试探、布局提供低成本启动计策"先锋令"：首回合移动力+1
成长期3-5回合建立优势累加型计策开始发力"连战"：每回合伤害+10%，可叠加
爆发期6-8回合收割、决胜释放积累的优势"总攻"：根据已造成伤害追加伤害

2. 反馈设计：让成长"看得见、摸得着"

📊 多层反馈系统

战中成长必须在视觉、数值、音效三个层面给玩家明确反馈：

👁️ 视觉反馈

  • 层数指示器：角色头顶显示当前叠加层数
  • 状态光环：成长型计策激活时角色发光
  • 伤害数字放大：随着成长，伤害数字逐渐变大/变色
  • 环境变化：环境计策改变战场视觉（如浓雾弥漫）

📈 数值反馈

  • 实时伤害预览：显示"下一击将造成XXX伤害"
  • 成长进度条：显示距离下一层还有多远
  • 优势对比：显示当前vs开局的数值差距
  • 结算统计：战后显示"本次战斗成长幅度"

🔊 音效反馈

  • 层数叠加音：每次叠加都有递增的音效
  • 爆发音效：高成长状态下释放技能的独特音效
  • 环境音效：环境计策改变时的场景音效
  • 胜利凯歌：根据成长幅度变化的结算音乐

3. 认知负荷：简化的决策，深度的策略

⚠️ 战棋游戏的认知负荷陷阱

铃兰之剑复杂度4.8分，但新手留存率最高的原因是渐进式复杂度。计策系统必须避免"信息过载"。

设计原则具体做法反例（应避免）
渐进解锁新手期只有3个基础计策，随进度解锁一开始就展示20个计策
智能推荐系统高亮当前最优计策选择让玩家完全自己摸索
效果可视化使用前显示预计效果范围/数值使用后才知效果
撤销/重置允许撤销未确认的操作一步错全盘输
模式识别计策图标颜色对应类型（红攻击/蓝防御/绿辅助）所有计策图标风格统一

4. 情感曲线：从焦虑到掌控

💡 战中成长的情感设计

优秀的计策系统应该让玩家经历这样的情感曲线：

  1. 开局焦虑："敌方很强，我该怎么办？" → 提供启动计策降低焦虑
  2. 中期掌控："我的计策开始起效了" → 成长反馈建立掌控感
  3. 后期爽感："看我一击必杀！" → 积累的优势转化为爆发爽感
  4. 战后满足："这一把打得太爽了" → 成长统计强化成就感

第一章：计策分类框架

基于SRPG公约数分析（再动15.8%、治疗20.8%、AOE39.3%覆盖率），将计策分为三大类别，与行动顺序、战场环境、数值状态三个维度对应：

⚡ 时机型计策

改变行动顺序，打破回合制节奏

  • 再行动（最高价值）
  • 行动提前/延后
  • 额外行动点
  • 跳过敌方回合

SRPG公约数 再动覆盖率15.8%，但战略价值极高

🌊 环境型计策

改变战场物理/规则条件

  • 召唤地形（水域/火焰）
  • 障碍物生成/清除
  • 天气/迷雾效果
  • 传送门/升降台

护城河机制 铃兰之剑环境互动是核心差异化

🔢 数值型计策

临时修改战斗数值参数

  • 伤害倍率调整
  • 移动力增减
  • 射程改变
  • 免疫/反制特定效果

SRPG公约数 最基础但风险最高的类型

📊 设计建议：三类计策的黄金比例

基于四款SRPG的复杂度分析（铃兰4.8分最高），建议计策池构成：

计策类型推荐占比设计难度玩家感知度
时机型20-25%高极高
环境型35-40%中高
数值型35-40%低中

理由：时机型（如再动）虽价值最高但需严格控制数量避免破坏节奏；环境型是差异化核心，可形成"护城河"；数值型作为基础填充，保证多样性。

耦合逻辑：

💡 耦合设计价值

  1. 教学意义：思考领域是"计策使用教程"，让玩家在无代价环境中掌握技巧
  2. 叙事融合：计策不再是抽象技能，而是"从角色思维中学习的能力"
  3. 收集驱动：通关思考领域解锁新计策，形成收集目标
  4. 复用设计：同一套机制（环境效果）在关卡和计策中复用，减少开发成本

第五章：核心设计观点

🎯 观点零：计策的核心是「战中成长」

这是与传统SRPG最本质的差异。传统战棋是静态博弈——开局阵容决定80%胜负；我们的计策系统应该创造动态成长——让玩家在战斗中持续变强，创造变数。

战中成长 = 数值累加型成长 + 优势造成型成长 + 适应型成长

📈 观点一：数值累加型成长

通过持续行动叠加数值，后期爆发。参考天地劫司马懿"鹰视狼顾"的设计。

  • 每次攻击叠加1层，最高5层
  • 每层+10%伤害，满层额外效果
  • 让玩家看到"我在越打越强"

例："战意"计策——每回合+15%攻击，可叠加3层

🌊 观点二：优势造成型成长

通过计策改变战场条件，滚雪球扩大领先。参考铃兰之剑环境互动。

  • 召唤地形限制敌方走位
  • 创造局部优势区域
  • 让优势像雪球一样越滚越大

例："诱敌深入"——敌人进入指定区域后受到伤害+50%

🔄 观点三：适应型成长

根据战场变化动态调整，越打越顺的掌控感。

  • 血量越低伤害越高（背水一战）
  • 敌方越少移动力越高（收割）
  • 根据敌方类型自动切换效果

例："绝境反击"——每损失10%生命，伤害+8%

⚖️ 观点四：限制即乐趣

基于SRPG数据分析，有限资源下的决策比无限使用更有乐趣。

  • ⏱️ CD限制：选择最佳时机
  • 🔋 能量限制：前戏积累的节奏感
  • 💰 代价限制：高风险高回报
  • 🎯 目标限制：特定条件触发

天地劫T0角色普遍有"条件再动"而非"无条件再动"

🌟 观点五：环境是差异化核心

数值计策易同质化，环境计策才是产品护城河。铃兰之剑复杂度4.8分的核心：环境交互机制占比最高。

❌ 同质化设计

+20%攻击（所有游戏都有）

✅ 差异化设计

召唤浓雾让刺客隐身（独特体验）
