# 军师系统设计方案 - 五位军师完整配置

> 基于计策设计案的军师系统完整方案
> 状态：草稿 | 创建时间：2026-03-16

---

## 系统概述

### 核心定位
军师不上场作战，通过**天赋**和**三计策**推动团队战中成长。

### 框架原则
1. **不上场**：军师不进入战场格子
2. **一天赋**：每位军师只有一个核心天赋（10%/30%/50%三级）
3. **三计策**：强制搭配（1时机型 + 1环境型 + 1数值型）
4. **中成长**：天赋和计策随战局演化而强化

## 军师一：诸葛亮（智慧型）

### 核心天赋：运筹帷幄

| 等级 | 触发条件 | 效果 |
|------|---------|------|
| **10%** | 开局即生效 | 每回合开始时，友方全体智力+5%，持续1回合 |
| **30%** | 使用3次计策后解锁 | 智力加成提升至15%，且附加「计谋」标记（受到计策伤害+10%） |
| **50%** | 友方有人阵亡时触发 | 立即触发「空城」：全体无敌1回合，免疫所有伤害（每战1次） |

**成长曲线**：缓启动 → 中期稳定增益 → 绝境保命

### 三计策配置

**1. 时机型：东风祭**
- **触发**：第3/6/9回合自动释放
- **效果**：全场友方下次攻击伤害+30%
- **成长**：每释放1次，伤害加成+10%（最高+60%）
- **CD**：自动触发，无CD

**2. 环境型：八阵图**
- **效果**：在指定区域生成八卦阵，友方在阵内闪避+25%，敌方命中率-20%
- **成长**：阵法持续回合+1（初始2回合，最高4回合）
- **范围**：5格菱形
- **CD**：4回合

**3. 数值型：空城计**
- **效果**：消耗50%当前生命，对敌方全体造成智力×1.5倍伤害
- **成长**：生命消耗比例降低（50%→40%→30%）
- **特殊**：当己方人数少于敌方时，伤害翻倍
- **CD**：5回合

### 军师-武将羁绊

| 武将 | 羁绊名称 | 效果 |
|------|---------|------|
| 赵云 | 鱼水之交 | 赵云在八阵图内时，攻击必定暴击 |
| 姜维 | 继承遗志 | 姜维使用计策时，50%概率不进入CD |
| 刘备 | 三顾之恩 | 刘备受到致命伤害时，诸葛亮自动释放空城（额外1次） |

## 军师二：司马懿（谋略型）

### 核心天赋：深谋远虑

| 等级 | 触发条件 | 效果 |
|------|---------|------|
| **10%** | 开局即生效 | 每回合开始时，获得1点「谋略」标记（最多5点） |
| **30%** | 谋略达到3点时解锁 | 每点谋略使友方全体伤害+3%（最高+15%） |
| **50%** | 敌方使用大招时触发 | 立即触发「鹰视狼顾」：偷取敌方全部怒气/能量，并反弹50%伤害 |

**成长曲线**：积累型 → 中期爆发 → 反制敌方核心

### 三计策配置

**1. 时机型：鹰视狼顾**
- **触发**：当敌方释放强力技能时自动触发
- **效果**：免疫该技能效果，并反弹30%伤害给施法者
- **成长**：反弹比例提升（30%→40%→50%）
- **CD**：每3回合可触发1次

**2. 环境型：上方谷**
- **效果**：在指定区域引发火攻，敌人每回合损失10%生命，持续3回合
- **成长**：火攻伤害叠加（敌人停留在火海中，伤害每回合+5%）
- **范围**：直线5格
- **CD**：4回合

**3. 数值型：隐忍待发**
- **效果**：本回合不行动，下回合伤害×2.5
- **成长**：伤害倍率提升（2.5→2.8→3.0）
- **特殊**：可与友方增益叠加
- **CD**：3回合

### 军师-武将羁绊

| 武将 | 羁绊名称 | 效果 |
|------|---------|------|
| 张春华 | 夫妻同心 | 张春华攻击时，司马懿额外获得1点谋略 |
| 司马昭 | 父子传承 | 司马昭击杀敌人时，谋略标记上限+1（临时） |
| 邓艾 | 宿敌对决 | 邓艾在场时，司马懿天赋成长速度+50% |

## 军师三：周瑜（火攻型）

### 核心天赋：赤壁业火

| 等级 | 触发条件 | 效果 |
|------|---------|------|
| **10%** | 开局即生效 | 火属性伤害+20%，火海持续时间+1回合 |
| **30%** | 造成5次火属性伤害后解锁 | 火属性伤害+40%，且火海会蔓延到相邻格子 |
| **50%** | 同时存在3个火海时触发 | 触发「火烧连营」：全场火海爆发，造成智力×2.0倍AOE伤害 |

**成长曲线**：火伤专精 → 火势蔓延 → 全场爆发

### 三计策配置

**1. 时机型：苦肉计**
- **触发**：当友方生命低于30%时
- **效果**：该友方攻击+50%，且攻击附加「灼烧」效果（每回合损失5%生命，持续3回合）
- **成长**：生命阈值提升（30%→40%→50%）
- **CD**：每战2次

**2. 环境型：火烧连营**
- **效果**：在指定区域引发大火，敌人每回合损失15%生命，移动力-2
- **成长**：火海范围扩大（3格→4格→5格圆形）
- **特殊**：与已有火海相邻时，会蔓延连接
- **CD**：3回合

**3. 数值型：周郎顾曲**
- **效果**：对敌方单体造成智力×1.8倍伤害，若目标处于火海中，伤害×2.5
- **成长**：基础倍率提升（1.8→2.0→2.2）
- **特殊**：击杀目标时，立即刷新CD
- **CD**：3回合

### 军师-武将羁绊

| 武将 | 羁绊名称 | 效果 |
|------|---------|------|
| 黄盖 | 苦肉主角 | 黄盖使用苦肉计时，生命消耗减半，伤害翻倍 |
| 小乔 | 琴瑟和鸣 | 小乔在场时，周瑜所有计策CD-1 |
| 孙策 | 江东双璧 | 孙策在火海中战斗时，攻击+30%且免疫灼烧 |

## 军师四：郭嘉（奇谋型）

### 核心天赋：十胜十败

| 等级 | 触发条件 | 效果 |
|------|---------|------|
| **10%** | 开局即生效 | 每回合开始时，随机揭示1个敌方单位的弱点（受到特定类型伤害+20%） |
| **30%** | 累计揭示5个弱点后解锁 | 揭示弱点数量+1，且弱点伤害加成+40% |
| **50%** | 友方攻击弱点5次后触发 | 触发「遗计」：郭嘉牺牲自己，全体友方立即行动1次（每战1次） |

**成长曲线**：情报收集 → 弱点放大 → 终极牺牲

### 三计策配置

**1. 时机型：遗计定辽东**
- **触发**：郭嘉阵亡时自动触发
- **效果**：全体友方攻击+50%，持续3回合
- **成长**：增益持续时间+1（3→4→5回合）
- **特殊**：可与主动释放叠加
- **CD**：每场战斗1次（被动）

**2. 环境型：天妒英才**
- **效果**：在指定区域生成「天命」领域，友方在领域内暴击率+30%，但受到伤害+10%
- **成长**：负面效果降低（+10%→+5%→0%）
- **范围**：4格菱形
- **CD**：4回合

**3. 数值型：算无遗策**
- **效果**：对敌方单体造成智力×1.5倍伤害，若击中弱点，伤害×3.0且眩晕1回合
- **成长**：弱点倍率提升（3.0→3.3→3.5）
- **特殊**：必定命中弱点单位
- **CD**：3回合

### 军师-武将羁绊

| 武将 | 羁绊名称 | 效果 |
|------|---------|------|
| 曹操 | 君臣相知 | 曹操攻击被郭嘉揭示弱点的敌人时，必定暴击 |
| 荀彧 | 王佐之才 | 荀彧在场时，郭嘉每回合额外揭示1个弱点 |
| 典韦 | 以命相护 | 郭嘉受到致命伤害时，典韦自动挡下（若典韦在场） |

## 军师五：曹操（统帅型）

### 核心天赋：雄才大略

| 等级 | 触发条件 | 效果 |
|------|---------|------|
| **10%** | 开局即生效 | 友方每次击杀敌人，全体攻击+3%（可叠加，最高15%） |
| **30%** | 累计击杀5个敌人后解锁 | 攻击加成上限提升至30%，且击杀回复10%生命 |
| **50%** | 己方人数少于敌方时触发 | 触发「官渡之战」：全体伤害+50%，受到伤害-30%，持续3回合 |

**成长曲线**：顺风滚雪球 → 均势强化 → 逆风翻盘

### 三计策配置

**1. 时机型：唯才是举**
- **触发**：当有新敌人入场时
- **效果**：立即对新人场敌人造成全体友方平均攻击力的伤害
- **成长**：伤害倍率提升（1.0→1.2→1.5倍）
- **特殊**：若击杀该敌人，立即再触发1次
- **CD**：每3回合可触发1次

**2. 环境型：挟天子令诸侯**
- **效果**：在战场中央生成「王旗」，友方在王旗周围3格内攻击+20%，敌方攻击-10%
- **成长**：王旗范围扩大（3格→4格→5格）
- **特殊**：王旗存在时，曹操天赋成长速度+50%
- **CD**：5回合（王旗持续3回合）

**3. 数值型：官渡之战**
- **效果**：对敌方全体造成攻击×0.8倍伤害，对满血敌人伤害翻倍
- **成长**：基础倍率提升（0.8→0.9→1.0）
- **特殊**：每击杀1个敌人，回复10%怒气/能量
- **CD**：4回合

## 系统设计规则

### 一、军师数量控制

- **军师:武将比例** = 1:8 ~ 1:10
- 50个武将的游戏，设计5-6个军师
- 确保每个军师都有不可替代性

### 二、计策限制体系

| 限制类型 | 规则 | 说明 |
|---------|------|------|
| **CD限制** | 小计策2-3回合，大计策4-5回合 | 防止滥用 |
| **能量限制** | 每次使用计策消耗20-50点谋略值 | 军师每回合恢复20点 |
| **代价限制** | 强力计策有生命/负面效果代价 | 高风险高回报 |
| **条件限制** | 部分计策需要特定站位/状态 | 增加策略深度 |

### 三、成长可视化

**军师面板UI**：
```
┌─────────────────────────────┐
│ 军师：诸葛亮 ★★★★★        │
│ 天赋：运筹帷幄              │
│ [▓▓▓▓▓▓░░░░] 60%          │
│                             │
│ 计策1：东风祭 [▓▓▓░░] Lv2  │
│ 计策2：八阵图 [▓▓▓▓░] Lv3 │
│ 计策3：空城计 [▓▓░░░] Lv1 │
│                             │
│ 谋略值：80/100              │
└─────────────────────────────┘
```

### 四、军师-武将联动

**羁绊触发提示**：
- 战斗开始时显示可用羁绊
- 羁绊触发时屏幕中央弹出特效
- 羁绊升级时有特殊动画

## UX设计说明

### 一、战斗界面布局

```
┌──────────────────────────────────────┐
│  [武将1]  [武将2]  [武将3]  [武将4]   │  ← 战场
│                                      │
│         [战场格子 5×7]               │
│                                      │
│  [武将5]        [敌人]               │
├──────────────────────────────────────┤
│  军师：诸葛亮                        │
│  [计策1] [计策2] [计策3]  [天赋进度] │  ← 军师面板
└──────────────────────────────────────┘
```

### 二、成长反馈

**视觉反馈**：
- 天赋升级时，军师头像闪烁金色光芒
- 计策成长时，对应按钮显示升级箭头
- 羁绊触发时，相关武将头像连线高亮

**音效反馈**：
- 天赋升级：庄重编钟声
- 计策释放：对应类型音效（火攻=燃烧声，谋略=算盘声）
- 羁绊触发：特殊和弦

### 三、新手引导

**第一层**：
- 教程关强制使用诸葛亮
- 引导释放八阵图 → 释放东风祭 → 触发天赋

**第二层**：
- 展示不同军师差异（曹操滚雪球 vs 郭嘉情报战）
- 引导玩家根据阵容选择军师

**第三层**：
- 高级策略：羁绊搭配、计策combo
- 示例：周瑜+黄盖苦肉计+火烧连营combo

## 数值边界（基于621位数据）

| 设计要素 | 边界值 | 说明 |
|---------|--------|------|
| 天赋基础效果 | 10-15% | 不低于T1英雄白板 |
| 天赋成长上限 | 40-60% | 不超过T0英雄极限 |
| 计策基础倍率 | 单体≤1.8倍，AOE≤0.7倍 | 参考phase3设计案红线 |
| 计策成长上限 | 基础值×1.5倍 | 确保成长空间 |
| 羁绊加成 | 20-30% | 显著但不破坏平衡 |
| 成长速度 | 3-5回合达峰值 | 参考再动机制稀缺性 |

---

## 核心设计原则

1. **稀缺即价值**：军师数量少（10-15%），每个都有独特定位
2. **选择即策略**：三计策强制1+1+1搭配，避免极端build
3. **成长即反馈**：每回合都能看到进度，绝境有翻盘可能
4. **羁绊即深度**：特定组合有额外效果，鼓励阵容研究
5. **可视化即体验**：所有成长都有UI反馈，降低认知负担
