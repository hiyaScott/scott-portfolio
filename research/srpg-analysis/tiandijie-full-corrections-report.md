## 三、错误原因深度分析

### 3.1 根本原因

```
┌─────────────────────────────────────────────────────────────┐
│                      数据采集流程缺陷                         │
├─────────────────────────────────────────────────────────────┤
│  1. 单一来源依赖                                              │
│     └── 使用早期攻略网站，非官方Wiki                          │
│                                                               │
│  2. 缺乏版本控制                                              │
│     └── 未标注数据对应的游戏版本                              │
│     └── 角色升阶（极→绝）未同步                              │
│                                                               │
│  3. 未验证录入                                                │
│     └── 数据录入后无抽样核查                                  │
│     └── 无多源交叉验证流程                                    │
│                                                               │
│  4. 结构混淆                                                  │
│     └── 天赋、技能、大招边界模糊                              │
│     └── 联动角色数据可能来自测试服                            │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 具体表现

| 问题 | 案例 | 影响 |
|------|------|------|
| 天赋名与技能名混淆 | 林月如天赋="斩龙诀" | 核心机制完全丢失 |
| 属性简化 | 冰璃再动条件被省略 | 战术价值评估错误 |
| 版本未更新 | 燕明蓉品阶未同步 | 角色强度误判 |
| 符号混淆 | 炎/火属相不统一 | 属性克制计算错误 |
