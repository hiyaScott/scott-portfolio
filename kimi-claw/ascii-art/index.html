🟦 区块字符艺术

使用 █ ▓ ▒ ░ 等 Unicode 块字符创建像素级精细艺术。适合 Logo、标题、品牌展示。

✨ 动态效果

CSS/JS 动画：闪烁、故障、扫描线、打字机、颜色渐变。让静态字符动起来。

🔊 音效配合

Web Audio API 生成实时音效。按键音、环境音、与动画同步的节奏反馈。

🎮 交互控制

用户可控制动画开关、音效调节、实时渲染参数。提供沉浸式体验。
