🔍 Tavily 智能搜索

专为 LLM 设计的搜索工具，过滤广告干扰，返回结构化结果。Brave Search 的替代方案。

✨ 核心优势

  • 🎯 为 LLM 优化 - 返回结构化、干净的结果
  • 🚫 广告过滤 - 自动过滤推广内容
  • 📊 信息聚合 - 自动总结多个来源
  • 🇨🇳 国内镜像 - 支持国内网络访问

📊 与 Brave Search 对比

特性 Brave Search Tavily Search
设计目标 通用隐私搜索 专门为 AI 优化
结果格式 标准网页列表 结构化摘要
广告过滤 基础 深度优化
上下文理解 一般 专为 LLM 设计
国内访问 需要 API key 有国内镜像源
多源聚合 无 自动聚合多个来源

🚀 使用场景

场景1: 快速事实核查

问: "OpenAI 最新发布的模型是什么？"
→ Tavily 搜索 → 返回最新准确信息

场景2: 深度行业研究

问: "分析2024年跨境电商市场的主要趋势"
→ Tavily 深度搜索 → 多源信息聚合 → 结构化报告

场景3: 技术文档查询

问: "Python 3.12 的新特性"
→ Tavily 搜索官方文档 + 社区文章 → 整理特性列表

⚡ 使用方法

基础搜索：
搜索: "关键字"
tavily: "查询内容"

高级搜索：
深度搜索: "主题" - 启用研究模式
搜索最新: "主题" - 优先最新结果
搜索中文: "主题" - 优先中文来源

⚠️ 注意事项

  • 🔑 API 限制 - 免费版有每日调用限制
  • 🔒 隐私 - Tavily 不记录个人搜索历史
  • ✅ 准确性 - 仍需人工核实关键信息
  • 📎 引用 - 使用 Tavily 结果时建议保留来源链接
