技术实现

Cron 任务系统

  • 调度器：OpenClaw Cron
  • 执行模式：isolated agentTurn（独立会话）
  • 容错：自动重试、超时处理
  • 日志：执行记录持久化存储

Heartbeat 机制

  • 每30分钟一次心跳检查
  • 批量检查任务状态
  • 异常情况主动告警

管理操作

  • cron list - 查看所有定时任务
  • cron add - 添加新任务
  • cron update - 更新任务
  • cron remove - 删除任务
  • cron run - 立即执行任务
