🧬 能力进化器 (CapabilityEvolver)

自我进化引擎，分析运行时历史以识别改进点，让 AI Agent 越用越聪明。

"装了不会立刻有变化，但用一两周后，它越来越懂你。你说一半它就知道你要什么，给出的结果和你期望越来越接近。属于「装了不后悔，用得越久越值」的类型。"

—— ClawHub 用户反馈

📈 核心数据

  • 🏆 ClawHub 下载量第一 - 35,000+ 次安装
  • 🧠 元技能 (Meta-Skill) - 不是添加特定能力，而是让 AI 自我进化
  • ⚡ 最新版本 - v1.39.0 (2026-03-25)

🔄 GEP 基因进化协议

1. 观察 (Observe) → 扫描会话历史 (~/.openclaw/agents/*.jsonl)
2. 分析 (Analyze) → 识别高频任务、重复错误、低效模式
3. 生成 (Generate) → 自动创建新的 Skill 或优化现有流程
4. 沉淀 (Preserve) → 将优化固化为可复用的"基因资产"
5. 应用 (Apply) → 动态注入到后续会话中

🎯 核心能力

1. 任务模式识别

识别你每天重复做的任务，发现工作习惯和偏好：

  • 发现每天早上9点检查邮件 → 自动生成"晨间邮件简报"skill
  • 发现经常把网页内容转成 Markdown → 自动生成"网页转笔记"skill

2. 错误自动修复

  • 记录你纠正 AI 的方式 → 自动记住偏好
  • 分析错误根本原因 → 生成补丁防止再次犯错

3. 个性化优化

  • 调整回复长度和风格
  • 优化工具调用顺序
  • 学习你的行业术语

💡 真实案例

某用户两周内，CapabilityEvolver 自动生成了 7 个新 Skill：

  • 行业简报自动整理
  • 客户沟通记录归档
  • 代码审查清单生成
  • 会议纪要格式化
  • 待办事项优先级排序
  • 文档版本对比
  • 周报自动生成

这些 Skill 不是用户让它做的，是它自己观察用户的工作习惯后主动生成的。

⚡ 使用方法

手动触发进化：
/evolve

强制重新分析：
/evolve --force

预览变更（不应用）：
/evolve --dry
