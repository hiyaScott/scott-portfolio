- **技术名称**：字符密度灰度映射（Character Density Grayscale Mapping）

## 三、第二段【动】：AI世界（50-90秒）

### 场景7：定格（80-90秒）【终局 - 高写实拉远】

**画面**：高写实合成AI脸定格，然后急速拉远

**过程**：
- 80-85秒：**合成脸凝视镜头，嘴角似乎上扬**（ASCII字符的微妙密度变化）
  - 高写实面部细节：能看到"皮肤纹理"的字符密度渐变

## 四、技术实现要点

### 1. 摄像头实时处理
```javascript
// 获取摄像头
navigator.mediaDevices.getUserMedia({ video: true })
  .then(stream => {
    video.srcObject = stream;
    // 开始渲染循环
    renderAsciiFrame();
  });

// 渲染循环（每3帧处理1次）
let frameCount = 0;
function renderAsciiFrame() {
  frameCount++;
  if (frameCount % 3 === 0) {
    // 绘制到canvas
    ctx.drawImage(video, 0, 0, 80, 60);
    // 获取像素数据
    const imageData = ctx.getImageData(0, 0, 80, 60);
    // 转换为ASCII
    const ascii = pixelsToAscii(imageData.data);
    // 显示
    asciiContainer.textContent = ascii;
  }
  requestAnimationFrame(renderAsciiFrame);
}
```

### 2. 字符密度映射
```javascript
const chars = '█▓▒░@#*+=:. ';
function pixelsToAscii(data) {
  let result = '';
  for (let i = 0; i < data.length; i += 4) {
    const brightness = (data[i] + data[i+1] + data[i+2]) / 3;
    const charIndex = Math.floor((brightness / 255) * (chars.length - 1));
    result += chars[charIndex];
    if ((i / 4 + 1) % 80 === 0) result += '\n';
  }
  return result;
}
```

### 3. 红色UI元素
```css
.red-ui {

  text-shadow: 0 0 10px #ff0000;
}
.blink {
  animation: blink 1s infinite;

@keyframes blink {
  0%, 50% { opacity: 1; }

}
```

### 4. 雪花噪点效果
```javascript
function generateNoise() {
  let noise = '';
  for (let i = 0; i < 80 * 60; i++) {
    noise += chars[Math.floor(Math.random() * chars.length)];
    if ((i + 1) % 80 === 0) noise += '\n';
  }
  return noise;
}
```

## 五、用户体验优化

### 1. 隐私提示
- 开始前明确显示"摄像头数据仅在本地处理，不会上传服务器"
- 提供"演示模式"（使用预设视频代替实时摄像头）

### 2. 性能优化
- 分辨率：80×60字符（手机友好）
- 帧率：10fps（每3帧渲染1次）
- 降级方案：如果性能不足，自动降低分辨率

### 3. 情感节奏
- 开始：游戏感、科技感（诱导）
- 中间：仪式感、掌控感（操控）
- 结尾：不安感、恐惧感（揭示）
- 片尾：虚无感、循环感

### 4. 无障碍
- 所有语音都有字幕
- 按钮足够大（手机触摸友好）
- 高对比度（黑白+红色）

## 七、风险评估

| 风险 | 概率 | 应对方案 |
|:---|:---:|:---|
| 用户拒绝摄像头权限 | 30% | 提供"演示模式"，使用预设视频 |
| 手机性能不足卡顿 | 20% | 自动降低分辨率，或跳过实时渲染直接播放预渲染 |
| 用户感到被冒犯 | 10% | 开始前明确说明是艺术创作，非真实数据采集 |
| 浏览器不兼容 | 5% | 检测API支持，不支持时显示静态版本 |
