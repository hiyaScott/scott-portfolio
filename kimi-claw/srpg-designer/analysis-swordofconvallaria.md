# 《铃兰之剑》战斗系统深度分析

## 核心特色

### 像素HD风格
- 复古像素画面
- HD-2D光影效果
- 高质量角色立绘

### 单机+网游混合模式
- 单机模式：买断制，完整剧情
- 网游模式：角色收集养成
- 两种模式独立但互通

## 战斗系统

### 职业克制

```
粉碎者 → 守护者 → 突袭者 → 粉碎者
毁灭者 ↔ 守望者（互克）
```

额外维度：力量 → 技巧 → 智慧 → 力量

### 地形机制

| 地形效果 | 数值 |
|---------|------|
| 高打低 | 攻击力+15% |
| 低打高 | 攻击力-10% |
| 背击 | +30%伤害 |
| 坠落伤害 | 根据高度差 |

### 场景互动

**可互动元素**：
- 炸药桶：范围高伤害
- 滚石：直线击退+伤害
- 悬崖：击退直接秒杀

### 技能类型

- 普攻：基础攻击
- 主动技能：消耗技能点（初始3点，行动后+1）
- 被动技能（个性）：角色固有能力
- 瞬发技能：立即使用，不消耗回合
- 光环技能：持续增益队友

## Roguelike元素

### 多周目设计
- 每次周目体验不同
- 随机招募角色
- 随机获得装备
- 多结局分支

### 养成系统
- 等级提升
- 个性培养（碎片升星）
- 技能强化
- 装备打造（强化/突破/升星/刻印）

## 战术系统

独立于回合外的指挥技能：
- 弱点解析：降低敌人防御
- 集火指令：击败目标获得再动
- 隐藏爆桶：远程引爆

## 设计亮点

1. **Roguelike+战棋**：高重玩价值
2. **地形杀**：利用环境取胜
3. **指挥官系统**：主角不亲自上场，专属战术技能
4. **像素美学**：复古与现代的结合
5. **单机体验**：买断制完整剧情
