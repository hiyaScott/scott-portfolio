🗺️ 专业知识地图

文档设计

Vision → High Concept → GDD → PBS → 详细文档。掌握从概念到实现的完整文档流程。

服务器架构

Client → Proxy → Game Logic → Database。理解四层架构与分区/分服/实例化策略。

经济循环

产出 → 存量 → 消耗 → 外部注入。掌握水龙头-排水管模型与通胀控制。

社交层次

个人 → 双向 → 小团体 → 大团体 → 全服。构建完整的社交关系金字塔。

玩法循环

行动 → 反馈 → 奖励 → 成长 → 新行动。设计心流体验与有意义的决策。

数值体系

战斗数值 + 经济数值 + 成长数值。建立完整的数值模型与验证流程。
