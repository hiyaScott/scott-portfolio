影视剧本基础

三幕结构

幕 页数 核心内容
第一幕 (建置) p1-p30 介绍人物、世界、日常；第一幕末出现激励事件
第二幕 (对抗) p30-p90 主角追求目标，遭遇挫折；p60为中点；第二幕末陷入最低谷
第三幕 (解决) p90-p120 高潮对决；解决主要冲突；p110-112为高潮；余韵结尾

剧本格式规范

INT. 咖啡馆 - DAY


李明坐在角落的位置，面前摆着一台笔记本电脑。 他盯着屏幕，手指悬在键盘上。


李明 (30岁)

deadline 是明天，我连一半都没做完。

角色塑造：Want vs Need

  • Want (欲望) - 角色表面上想要的东西（外在目标）
  • Need (需求) - 角色内心深处真正需要的东西（内在成长）
  • 人物弧光 - 角色在故事中的成长或转变

游戏叙事设计

分支叙事结构

珍珠串结构

主线A → [支线B] → 主线A → [支线C] → 主线A

线性主干 + 可选分支，分支不影响主线

分支与瓶颈

路线A/B/C → 汇合点 → 结局

玩家可选择不同路径，但最终汇合到主线

开放世界

完全非线性结构

玩家自由探索，从不同角度接近故事节点

玩家自主权 (Agency)：
玩家需要被置于体验的中心。选择必须在关键时刻产生实质影响，避免"假选择"。

阶段 人类负责 AI负责

结构规划 确定关键情节点、转折点位置 生成完整节拍表、多版本结构方案

修改润色 风格统一、艺术判断 检查一致性、语法、格式、节奏分析

AI擅长 vs 人类必须亲自写

✅ AI擅长

  • 结构分析 - 检查节拍分布、识别pacing问题
  • 角色一致性检查 - 追踪动机、行为模式是否连贯
  • 场景扩展 - 基于大纲生成动作描述、环境描写
  • 对白选项生成 - 同一情境下的多种对白版本
  • 格式整理 - 自动调整剧本格式、分页
  • 研究辅助 - 时代背景、专业术语、文化细节

❌ 人类必须亲自写

  • 核心创意 - 独特世界观、核心隐喻
  • 情感真实 - 角色内心挣扎的真实描写
  • 文化洞察 - 特定文化的深层含义
  • 个人声音 - 独特的叙事风格和作者印记
  • 高潮设计 - 情感高潮的精准把控

Laper AI（国产剧本AI）

官网：laper.ai

核心功能：

  • AI节拍生成 - 输入一句话概念，自动生成Save the Cat节拍表
  • 角色弧线追踪 - 可视化展示角色从第一幕到第三幕的转变
  • Pacing分析 - 识别剧本节奏问题，标注拖沓或过于紧凑的段落
  • 情感曲线映射 - 追踪观众情绪起伏，优化情感节奏

使用流程：

  1. 导入概念文档（支持.txt, .fdx格式）
  2. 选择剧本类型（喜剧/剧情/惊悚等）
  3. 生成节拍表（提供3个版本供选择）
  4. 逐场检查pacing建议
  5. 导出Final Draft兼容格式

📝 Prompt工程模板

黄金公式

[场景类型] + [情绪基调] + [核心冲突] + [特殊要求]

高效提示 vs 低效提示

❌ 低效提示：
"写一个办公室场景"

✅ 高效提示：
"写一个紧张压抑的办公室场景，主角发现同事偷窃创意，需要决定是否揭发。要求：通过环境细节暗示主角的焦虑，避免直接心理描写，使用现在时态和主动语态。"

"规则三"技巧

让AI生成3个选项再选择，避免"首因效应"，提供对比空间：

应用模板：

"请为[任务]生成3个选项：
选项1：传统/经典处理方式
选项2：反套路/意外处理方式  
选项3：介于两者之间的折中方案

每个选项附带简短说明其优缺点。"

角色声音训练模板

样本训练法：

  1. 提供角色背景（年龄、职业、性格）
  2. 提供3-5句样本对白
  3. 要求基于风格生成新情境下的对白

示例：

角色：老张，50岁，退休刑警，性格固执但内心柔软

样本："你这案子，我干了三十年，闻都能闻出味儿来。"

任务：基于以上风格，写老张得知女儿被绑架后的反应，3个不同版本

AI结构分析提示词

"作为剧本结构分析师，请分析以下场景列表： [粘贴场景列表] 请提供： 1. 每幕的页数分布是否合理？ 2. 识别3个最可能拖沓的场景及原因 3. 指出缺少"呼吸空间"的段落 4. 建议2处可以增加反转的位置"
